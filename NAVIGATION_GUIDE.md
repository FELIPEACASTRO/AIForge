## Conventions

- **Directory naming**: `Snake_Case_English_Only`. No spaces, accents, or commas.
- **No duplicates**: a topic exists in exactly one place.
- **Cross-cutting**: when a topic touches multiple pillars (e.g., RAG datasets), it lives in the *primary* pillar and is *linked* from others' READMEs.
- **READMEs**: every new subtree has a README with description and references.
