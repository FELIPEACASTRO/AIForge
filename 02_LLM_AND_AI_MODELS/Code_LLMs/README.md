# Code LLMs

> Code generation is one of the highest-value LLM verticals (Copilot, Cursor, agentic coding). The repo has no dedicated section: grep for 'starcoder|codellama|deepseek-coder|qwen-coder' returns 0 hits anywhere. Only a passing 'code llm' mention in the pillar README. The major code-specialized families (StarCoder2, Code Llama, DeepSeek-Coder, Qwen2.5-Coder, CodeGemma) warrant a standalone section separate from general Text_LLMs.
