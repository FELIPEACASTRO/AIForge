# LLM Guides and Tutorials

This directory covers practical guides for learning, evaluating, fine-tuning, deploying, and safely using LLMs and multimodal models.

## Scope

- Model selection, prompting, context windows, tool use, structured outputs, RAG, fine-tuning, evaluation, safety, and deployment.
- Provider guides, open-source tutorials, notebooks, reproducible walkthroughs, and migration notes.
- Beginner-to-advanced learning paths that are source-backed and dated.
