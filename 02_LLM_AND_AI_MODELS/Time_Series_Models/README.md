# ⏱️ Time Series Forecasting / Previsão de Séries Temporais

---

## 🌍 English

Foundation models and architectures for time series forecasting that achieve state-of-the-art performance with zero-shot and few-shot learning capabilities.

### 🎯 Overview

This category contains cutting-edge foundation models, architectures, benchmarks, and tutorials for time series forecasting. These resources represent the latest breakthroughs in using AI to predict future values across diverse domains including finance, energy, retail, healthcare, and more.

### 🏆 Key Breakthroughs

- **Zero-Shot Forecasting**: No task-specific training required
- **Few-Shot Learning**: Adapt to new domains with minimal data
- **Ultra Compact Models**: 35M parameters outperform 500M models
- **Foundation Models**: Pretrained on massive datasets, ready to use
- **Multi-Domain**: Single model works across finance, energy, retail, etc.

## 📂 Categories

### 01. Foundation Models
Pretrained models ready for zero-shot and few-shot forecasting:
- **TiRex** (NXAI) - 35M params, beats 500M models, zero-shot
- **Chronos-2** (Amazon) - Latest generation, multivariate support
- **Chronos-Bolt** (AutoGluon) - 4.37M downloads, fastest inference
- **TimesFM-2.5** (Google) - 200M params, few-shot learning
- **IBM Granite TTM** (IBM) - Ultra compact (800K params), 313 likes

### 02. Architectures
Novel architectures for time series:
- **xLSTM** - Enhanced LSTM (TiRex)
- **T5-based** - Transformer architecture (Chronos)
- **Fourier Neural Operator** - Frequency domain (FourCastNet)
- **Tiny Time Mixer** - Ultra compact (IBM Granite TTM)

### 03. Benchmarks
Standard benchmarks and leaderboards:

- **Chronos Benchmark**
