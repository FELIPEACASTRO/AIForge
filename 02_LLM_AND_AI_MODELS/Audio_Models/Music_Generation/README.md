# Music Generation

> Audio_Models contains only a Whisper (ASR) file. Generative music/audio is a major model family with zero dedicated coverage: 'musicgen|musiclm|suno|udio|stable audio' hits are all passing mentions in collection/multimodal/closed-model files, none a real section. MusicGen, MusicLM, Stable Audio, AudioLDM, Suno, Udio, YuE form a distinct generative-audio family separate from speech ASR.
