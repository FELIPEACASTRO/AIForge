# Vision Models

This directory covers AI models for images and visual perception.

## Scope

- Classification, detection, segmentation, OCR, depth, pose, tracking, image generation, restoration, and visual representation learning.
- Track architecture, checkpoint, dataset, license, input resolution, task head, benchmark, and deployment constraints.

## Reference Links

- TorchVision models: https://docs.pytorch.org/vision/main/models.html
- Hugging Face vision models: https://huggingface.co/models?pipeline_tag=image-classification
- timm documentation: https://huggingface.co/docs/timm/index
- OpenCV: https://opencv.org/

## Routing Rules

- Put VLMs in `../Multimodal_Models/` or the foundations VLM directory.
- Put application examples in computer-vision verticals.
