# AI Model Frameworks

This directory covers frameworks used to build, train, fine-tune, evaluate, export, and run AI/ML models.

## Scope

- Deep learning, classical ML, distributed training, fine-tuning, model export, model optimization, and inference runtimes.
- Framework notes should track language, license, model families, hardware support, distributed support, export formats, and production fit.
