# Model Catalog

This directory covers general AI model catalog entries that do not fit a narrower model-family folder yet.

## Scope

- Model cards, provider pages, checkpoints, APIs, licenses, benchmarks, context limits, input/output modalities, and deployment constraints.
- Use this as a temporary landing area only when the model's final family folder is unclear.

## Reference Links

- Hugging Face models: https://huggingface.co/models
- OpenAI models: https://developers.openai.com/api/docs/models
- Anthropic models: https://docs.anthropic.com/en/docs/about-claude/models/overview
- Google Gemini models: https://ai.google.dev/gemini-api/docs/models
- Papers with Code models: https://paperswithcode.com/methods

## Routing Rules

- Move text LLMs, video models, vision models, scientific models, and multimodal models into their dedicated folders once classified.
- Put benchmark-only notes in the benchmark directories.
