# AI Papers

This directory covers paper discovery, tracking, summaries, reading lists, citation maps, and source-backed research notes for AI, ML, data, LLMs, agents, and applied domains.

## Scope

- Research-paper indexes, paper digests, milestone papers, benchmark papers, model-release papers, and reproducibility notes.
- Metadata for title, authors, venue, date, code, data, license, benchmark, claims, limitations, and follow-up work.
- Separation of evidence from interpretation: cite the paper or official project page before deriving claims.

## Reference Links

- arXiv: https://arxiv.org/
- Semantic Scholar: https://www.semanticscholar.org/
- Papers with Code: https://paperswithcode.com/
- OpenReview: https://openreview.net/
- ACL Anthology: https://aclanthology.org/
- Hugging Face Papers: https://huggingface.co/papers

## Routing Rules

- Put model cards and provider documentation in the model family directories.
- Put dataset details in `../../03_DATASETS_TOOLS_AND_RESOURCES/Datasets/`.
- Put benchmark summaries in `../../03_DATASETS_TOOLS_AND_RESOURCES/Datasets/Famous_Benchmarks/` or evaluation folders.
