# Multimodal Models

This directory covers models that combine text, image, audio, video, structured data, tools, and other modalities.

## Scope

- Vision-language models, audio-language models, video-language models, document AI, multimodal embeddings, and multimodal generation.
- Track modalities, context limits, input/output formats, model card, benchmark evidence, safety controls, and licensing.
