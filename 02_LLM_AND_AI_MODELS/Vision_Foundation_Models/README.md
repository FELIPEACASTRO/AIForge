# Vision Foundation Models

> The repo's Vision_Models dir has only a 17-line SAM stub and a DiT file; 'DINOv2' appears once as a list item in a collection file. Self-supervised vision backbones (DINOv2, I-JEPA), open-vocabulary encoders (CLIP, SigLIP, SigLIP2), and the segmentation foundation family (SAM, SAM 2) are foundational and far thinner than they should be for a leading model index. A proper Vision_Foundation_Models section is needed.
