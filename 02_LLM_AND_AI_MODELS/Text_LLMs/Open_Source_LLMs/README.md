# Open Source and Open Weight LLMs

This directory covers open-source, open-weight, research, and community LLM families.

## Scope

- Llama, Mistral, Qwen, DeepSeek, Gemma, Phi, OLMo, Falcon, BLOOM, Yi, StarCoder, Code Llama, and other open or partially open model families.
- Track license, weights, training-data transparency, model card, context window, tokenizer, architecture, benchmark evidence, safety notes, quantizations, and serving compatibility.

## Reference Links

- Hugging Face models: https://huggingface.co/models
- Meta Llama: https://www.llama.com/
- Mistral model docs: https://docs.mistral.ai/getting-started/models/models_overview/
- Qwen: https://qwenlm.github.io/
- DeepSeek API docs: https://api-docs.deepseek.com/
- Google Gemma: https://ai.google.dev/gemma

## Routing Rules

- Put closed hosted models in `../Frontier_Closed_Models/`.
- Put fine-tuning and serving notes in MLOps or model-framework directories.
- Put model-evaluation evidence in benchmarks or evaluation folders.
