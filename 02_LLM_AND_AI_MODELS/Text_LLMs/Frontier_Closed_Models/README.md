# Frontier Closed Text LLMs

This directory covers closed or hosted frontier text LLMs, including their model families, APIs, capability notes, pricing considerations, context windows, tool use, safety controls, and evaluation evidence.

## Scope

- OpenAI, Anthropic, Google Gemini, Microsoft, xAI, Cohere, Mistral hosted models, and other closed frontier providers.
- Model cards, API docs, release notes, system limits, structured outputs, tool use, long context, multimodal adjacency, and eval caveats.
- Evidence-backed comparisons only; avoid unsourced leaderboard claims.
