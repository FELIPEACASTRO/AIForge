# Video Generation Models

This directory covers text-to-video, image-to-video, video-to-video, editing, extension, animation, and controllable video generation models.

## Scope

- Video diffusion, autoregressive, transformer, world-model, and multimodal generation systems.
- Track provider/model, input modalities, duration, resolution, audio support, editing controls, safety controls, watermarking, API access, pricing, and license.
