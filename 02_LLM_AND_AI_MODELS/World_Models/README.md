# World Models

Generative models that learn dynamics of the world for planning, simulation, and robotics.

## Foundational Work

- **World Models** (Ha & Schmidhuber, 2018) — https://worldmodels.github.io/
- **Dreamer / DreamerV2 / DreamerV3** (Hafner et al.) — https://github.com/danijar/dreamerv3
- **MuZero** (DeepMind, 2019) — https://arxiv.org/abs/1911.08265
- **PlaNet** (Hafner 2019) — https://arxiv.org/abs/1811.04551

## Recent Frontier (2024-2026)

- **Genie / Genie 2 / Genie 3** (DeepMind) — text/image-to-playable-world — https://deepmind.google/research/projects/genie/
- **NVIDIA Cosmos** — physical-AI World Foundation Models — https://www.nvidia.com/en-us/ai/cosmos/
- **GAIA-1 / GAIA-2** (Wayve) — autonomous driving world model — https://wayve.ai/thinking/gaia-1/
- **Sora** (OpenAI) — text-to-video as implicit world simulator
- **V-JEPA / V-JEPA 2** (Meta, LeCun) — Joint Embedding Predictive Architecture — https://github.com/facebookresearch/jepa
- **OASIS** (DecartAI) — playable Minecraft world model — https://oasis-model.github.io/
- **Lumiere** (Google) — text-to-video diffusion
- **Veo 2 / Veo 3** (DeepMind) — high-fidelity video generation

## Robotics World Models
