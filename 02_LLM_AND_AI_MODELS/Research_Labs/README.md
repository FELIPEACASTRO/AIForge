# AI Research Labs

This directory covers research labs and organizations that publish AI models, papers, datasets, benchmarks, and technical reports.

## Scope

- Academic, nonprofit, industry, government, and open-science AI labs.
- Entries should separate official lab pages, model releases, papers, datasets, and third-party interpretation.

## Reference Links

- OpenAI research: https://openai.com/research/
- Google DeepMind: https://deepmind.google/
- Meta AI: https://ai.meta.com/research/
- Allen Institute for AI: https://allenai.org/
- Hugging Face papers: https://huggingface.co/papers

## Routing Rules

- Put model-specific documentation in the model family folders.
- Put paper notes in `../Papers/`.
