# Financial Statement Data Sets

## Description
Os Financial Statement Data Sets (Conjuntos de Dados de Demonstrações Financeiras) fornecem informações numéricas extraídas das demonstrações financeiras de relatórios corporativos arquivados na Securities and Exchange Commission (SEC) dos EUA. Os dados são extraídos de relatórios financeiros corporativos arquivados na Comissão usando a linguagem eXtensible Business Reporting Language (XBRL). As informações são apresentadas sem alterações em relação aos relatórios financeiros "conforme arquivados" enviados por cada registrante. Os dados são apresentados em um formato achatado para ajudar os usuários a analisar e comparar as informações de divulgação corporativa ao longo do tempo e entre os registrantes.

## Statistics
Os dados estão disponíveis de janeiro de 2009 a junho de 2025 e são atualizados trimestralmente. O tamanho dos arquivos ZIP trimestrais varia, com os arquivos mais recentes (2023-2025) variando de aproximadamente 75 MB a 121 MB.

## Features
Dados numéricos de demonstrações financeiras; formato XBRL; atualizações trimestrais; inclui o Standard Industrial Classification (SIC) da empresa.

## Use Cases
Análise de demonstrações financeiras, comparação de desempenho de empresas, pesquisa acadêmica em finanças e contabilidade, desenvolvimento de modelos de previsão financeira e análise de tendências de mercado.

## Integration
Os conjuntos de dados podem ser baixados como arquivos ZIP trimestrais diretamente do site da SEC. Cada arquivo ZIP contém os dados em um formato de texto que pode ser importado para bancos de dados ou ferramentas de análise. A SEC também fornece documentação em PDF com detalhes sobre o escopo, organização, formatos de arquivo e definições de tabela. Existem também bibliotecas de código aberto (por exemplo, em Python) disponíveis para simplificar o processo de trabalho com esses dados.
