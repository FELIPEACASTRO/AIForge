# Quandl Financial Data (Nasdaq Data Link)

## Description
O Quandl, agora parte do Nasdaq Data Link, é uma plataforma de inteligência financeira baseada em nuvem que fornece acesso a uma vasta e diversificada coleção de dados financeiros, econômicos e alternativos. A plataforma unificada permite que investidores institucionais, fintechs e usuários individuais descubram, integrem e analisem dados de alta qualidade para otimizar estratégias de investimento e pesquisa. Embora o nome Quandl ainda seja usado para se referir à sua coleção de dados históricos, a infraestrutura e o acesso são fornecidos pelo Nasdaq Data Link.

## Statistics
A plataforma Nasdaq Data Link (antiga Quandl) indexa milhões de séries temporais de dados de mais de 400 fontes. O catálogo inclui mais de 250 conjuntos de dados premium e uma vasta coleção de dados gratuitos e abertos. O dataset WIKI Prices, um dos mais conhecidos, contém dados de fim de dia para 3.000 empresas dos EUA, mas foi descontinuado em 2018. A plataforma é utilizada por mais de 800.000 usuários globais.

## Features
Acesso a dados financeiros, econômicos e alternativos de alta qualidade; dados de séries temporais e tabelas (Datatables); API RESTful e Streaming API para dados em tempo real; curadoria rigorosa dos dados (menos de 1 em 100 datasets analisados se tornam produtos); integração nativa com Python, R, Excel e SQL; dados históricos e em tempo real.

## Use Cases
Modelagem e previsão de mercado; backtesting de estratégias de negociação quantitativa; pesquisa acadêmica em finanças e economia; análise de risco e conformidade; desenvolvimento de modelos de Machine Learning para análise de sentimento e previsão de preços de ações.

## Integration
O acesso é feito através da API do Nasdaq Data Link, que requer uma chave de API (gratuita para dados abertos). O Nasdaq Data Link oferece SDKs e bibliotecas para Python, R, Excel e SQL, facilitando a integração. O acesso aos dados é feito por meio de chamadas à API para séries temporais ou tabelas (Datatables).
