# Matterport3D

## Description
Matterport3D é um dataset RGB-D em larga escala projetado para o aprendizado de máquina em ambientes internos. Ele fornece dados de 90 cenas em escala de construção, capturados com uma câmera Matterport Pro. O dataset é rico em anotações, incluindo reconstruções de superfície, poses de câmera, e segmentações semânticas 2D e 3D. É uma referência fundamental para tarefas de visão computacional 3D.

## Statistics
Versão V1.0. Contém 90 cenas de edifícios, 10.800 vistas panorâmicas alinhadas e 194.400 imagens RGB-D. O tamanho total do download é de aproximadamente 200 GB (estimativa de fontes secundárias).

## Features
Dados RGB-D de alta resolução, malhas 3D texturizadas, plantas baixas de edifícios, anotações de regiões, e anotações semânticas de instâncias de objetos. O alinhamento global preciso e o conjunto diversificado de vistas panorâmicas sobre edifícios inteiros são características chave.

## Use Cases
Segmentação semântica, classificação de cenas, correspondência de pontos-chave (keypoint matching), previsão de sobreposição de vistas, estimação de normais de superfície, e tarefas de IA incorporada (Embodied AI), especialmente como base para datasets mais recentes como o Habitat-Matterport 3D (HM3D).

## Integration
O acesso ao dataset requer o preenchimento e assinatura do formulário 'Termos de Uso' e envio para matterport3d@googlegroups.com. Após a aprovação, o acesso para download é fornecido. O código e as ferramentas para visualização e processamento estão disponíveis no repositório GitHub oficial.
