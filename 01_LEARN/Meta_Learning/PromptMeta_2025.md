### Overview

Published in 2025, PromptMeta is a novel meta-learning framework for Few-Shot Relational Learning (FSRL) in Knowledge Graphs (KGs). It addresses a key limitation of previous FSRL methods, which focused only on relational structure while ignoring the rich semantics inherent in KGs.

### Key Innovations

- **Meta-Semantic Prompt (MSP) Pool:** Introduces a pool of prompts that learns and consolidates high-level meta-semantics shared across different tasks.
- **Prompt-Augmented Learning:** Augments the learning process with these meta-semantic prompts, enabling effective knowledge transfer and adaptation to new, unseen relations with very few examples.
- **State-of-the-Art Performance:** The paper claims to outperform existing state-of-the-art methods in few-shot relational learning.

### Impact

PromptMeta is a game-changer for Meta-Learning and Knowledge Graphs. By combining the power of prompting (popularized by LLMs) with meta-learning, it provides a powerful solution to the critical problem of data scarcity in KGs, enabling more robust and accurate reasoning over incomplete knowledge.
