# Projetos de Geração de Vídeo Open Source (VideoCrafter, CogVideoX, Open-Sora)

## Description

Open Source Video Generation Projects - Video synthesis, editing

## Statistics

Mais de 5.000 a 27.800 estrelas no GitHub. Modelos CogVideoX-2B e Open-Sora 2.0 otimizados para eficiência e acessibilidade de hardware. Suporte a vídeos de até 10 segundos e resoluções de até 768x768.

## Features

Geração de Texto para Vídeo (T2V), Geração de Imagem para Vídeo (I2V), Continuação de Vídeo, Modelos de Difusão de Vídeo de Alta Qualidade, Otimização para baixo consumo de memória de GPU, Suporte a fine-tuning com Lora, Suporte a inferência quantizada (INT8, FP8), Pipeline de T2V/I2V (T2I2V) otimizada, Suporte a múltiplas resoluções e proporções de aspecto, Avaliação de pontuação de movimento (Motion Score) e refinamento de prompt.

## Use Cases

Criação de conteúdo de vídeo de alta qualidade a partir de prompts de texto ou imagens estáticas, Pesquisa e desenvolvimento em modelos de difusão de vídeo, Geração de vídeos para mídias sociais e projetos criativos, Produção de vídeo de nível comercial com custo-benefício, Fine-tuning de modelos para domínios específicos.

## Integration

Projetos baseados em Python e PyTorch. Utilizam ambientes Conda e gerenciadores de pacotes como pip. A inferência é realizada por scripts shell ou Python, com modelos disponíveis no Hugging Face. O Open-Sora utiliza ColossalAI para paralelismo em multi-GPU. CogVideoX suporta estruturas SAT e Diffusers.
