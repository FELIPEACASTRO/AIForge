# COVID-19CT+

## Description

Conjunto de dados público de imagens de Tomografia Computadorizada (TC) para análise retrospectiva da COVID-19. Inclui múltiplos tipos de pneumonia, sendo um dos maiores datasets disponíveis, com foco em casos de COVID-19 e pneumonia adquirida na comunidade.

## Statistics

409.619 imagens de TC de 1333 pacientes. Subconjunto-A: 312 casos de pneumonia adquirida na comunidade. Subconjunto-B: 1021 casos de COVID-19.

## Features

Imagens de TC de alta resolução, cobrindo diferentes estágios da doença e tipos de pneumonia.

## Use Cases

Classificação de COVID-19 vs. pneumonia adquirida na comunidade, desenvolvimento e teste de algoritmos de diagnóstico automatizado (Machine Learning e Deep Learning).

## Integration

O artigo menciona o uso de 13 classificadores clássicos de Machine Learning e 5 classificadores de Deep Learning para testar a classificação de imagens, indicando a adequação para ambos os tipos de modelos. A URL primária aponta para o artigo, que deve conter o link de acesso ao dataset.
