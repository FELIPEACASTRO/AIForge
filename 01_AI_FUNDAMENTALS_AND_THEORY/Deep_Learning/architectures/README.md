# Deep Learning Architectures

This directory covers neural network architecture families and design patterns.

## Scope

- CNNs, RNNs, transformers, autoencoders, diffusion models, GNNs, state-space models, mixture-of-experts, retrieval-augmented models, and hybrid architectures.
- Track inductive bias, input modality, scaling behavior, compute profile, benchmark evidence, and failure modes.
