# Statistical Learning

This directory covers the statistical foundations used to build, validate, interpret, and compare machine-learning systems.

## Content Map

| Subdirectory | Scope |
|---|---|
| `Linear_Regression/` | Linear models, assumptions, residuals, regularization context, and baseline regression. |
| `Logistic_Regression/` | Classification, odds, calibration, decision thresholds, and interpretable baselines. |
| `Generalized_Linear_Models/` | GLMs for count, binary, continuous, and non-normal targets. |
| `Regularization_Ridge_Lasso_ElasticNet/` | Shrinkage, sparsity, feature selection, and bias-variance trade-offs. |
| `Resampling_and_Cross_Validation/` | Train/test splits, k-fold, bootstrap, leakage control, and model selection. |
| `Hypothesis_Testing/` | p-values, confidence intervals, multiple testing, and experiment interpretation. |
| `Experimental_Design_and_AB_Testing/` | Randomization, power, sample size, online tests, and causal measurement. |
| `Statistical_Time_Series/` | ARIMA-style models, decomposition, forecasting baselines, and temporal validation. |
| `Survival_Analysis/` | Time-to-event modeling, censoring, hazard models, and medical/industrial reliability use cases. |

## Enrichment Standard

Statistical pages should include assumptions, when the method fails, minimum data requirements, leakage risks, and a baseline example.
