# Transfer Learning

This directory covers transfer learning foundations across modalities and domains.

## Scope

- Fine-tuning, feature extraction, linear probing, adapters, LoRA, domain adaptation, self-supervised pretraining, few-shot learning, and continual adaptation.
- Track source domain, target domain, frozen layers, adaptation data, evaluation split, and domain-shift risks.

## Reference Links

- PyTorch transfer learning tutorial: https://docs.pytorch.org/tutorials/beginner/transfer_learning_tutorial.html
- TorchVision models: https://docs.pytorch.org/vision/main/models.html
- Hugging Face PEFT: https://huggingface.co/docs/peft/index
- timm documentation: https://huggingface.co/docs/timm/index

## Routing Rules

- Put healthcare transfer-learning applications in the healthcare transfer-learning folder.
- Put model-specific fine-tuning in the relevant model family folder.
