# Long Context Models

This directory covers models and methods for long-context reasoning, retrieval, memory, compression, and extended attention.

## Scope

- Long-context transformers, retrieval-augmented generation, context compression, chunking, memory systems, positional encoding, attention variants, and evaluation.
- Track context window, effective context, lost-in-the-middle behavior, latency, cost, retrieval policy, and factuality.
