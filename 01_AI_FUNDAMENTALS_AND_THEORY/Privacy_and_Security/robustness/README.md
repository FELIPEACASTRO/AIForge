# Robustness

This directory covers model reliability under adversarial examples, corruption, distribution shift, noise, spurious correlations, and operational stress.

## What Belongs Here

- definitions, assumptions, variants, and failure modes for the topic
- primary papers, official framework docs, benchmark evidence, and reproducible examples
- evaluation metrics, ablations, limitations, and notes on when the method should not be used
