# Privacy And Security

This directory covers privacy-preserving ML, secure AI systems, adversarial robustness, model security, data protection, and AI-specific attack surfaces.

## Content Map

| Subdirectory | Scope |
|---|---|
| `differential_privacy/` | DP-SGD, privacy budgets, composition, private analytics, and DP evaluation. |
| `robustness/` | Adversarial examples, certified defenses, distribution shift, and robustness testing. |

## Security Themes

- Data poisoning and backdoors.
- Model extraction, inversion, and membership inference.
- Prompt injection and tool-use attacks.
- Supply-chain risks in models, datasets, checkpoints, and agents.
- Privacy controls for sensitive and regulated data.

## Enrichment Standard

Record threat model, attacker capability, affected asset, mitigation, evaluation method, and residual risk.
