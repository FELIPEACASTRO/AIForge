# Knowledge Graphs

> Knowledge graphs (KG embeddings TransE/RotatE/ComplEx, KG completion, KG-augmented LLMs/GraphRAG) underpin retrieval and reasoning, but only appear incidentally in RAG/Meta-Learning docs. The GNN section covers message-passing but not KG embedding/reasoning. A distinct foundational topic that is currently missing.
