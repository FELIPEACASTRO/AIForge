# Machine Learning

This directory covers general machine-learning concepts that do not fit only into statistical learning, classical algorithms, or deep learning.

## Content Map

| Subdirectory | Scope |
|---|---|
| `Continual_Learning/` | Lifelong learning, catastrophic forgetting, rehearsal, regularization, and online adaptation. |
| `training_strategies/` | Curriculum learning, sampling, augmentation, optimization schedules, and training playbooks. |

## Core Topics

Use this directory for general ML workflows: problem framing, data splitting, bias-variance trade-offs, baseline design, evaluation discipline, model iteration, and failure analysis.

## Routing Rules

- Put algorithms in `../Classical_ML_Algorithms/`.
- Put neural methods in `../Deep_Learning/`.
- Put metrics and validation in `../Model_Evaluation/`.
