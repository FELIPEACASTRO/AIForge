# Vision Language Models

This directory covers foundations of models that connect visual and textual information.

## Scope

- CLIP-style contrastive models, image-text retrieval, visual question answering, captioning, OCR/document VLMs, grounding, and multimodal instruction tuning.
- Track image encoder, text encoder, alignment objective, pretraining data, zero-shot behavior, prompt sensitivity, and evaluation tasks.

## Reference Links

- OpenAI CLIP: https://openai.com/index/clip/
- Hugging Face CLIP docs: https://huggingface.co/docs/transformers/en/model_doc/clip
- Hugging Face VLM overview: https://huggingface.co/blog/vlms
- Hugging Face computer vision course VLM unit: https://huggingface.co/learn/computer-vision-course/en/unit4/multimodal-models/clip-and-relatives/Introduction
