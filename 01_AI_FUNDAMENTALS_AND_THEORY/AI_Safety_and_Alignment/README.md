# AI Safety and Alignment

Curated resources for **AI Safety**, **Alignment**, **Interpretability**, **Red-Teaming**, and **Robustness** research — covering the technical, governance, and policy stack from frontier labs (Anthropic, OpenAI, DeepMind, METR, MIRI) and academia.

## Core Subtopics

| Subtopic | Description |
|---|---|
| **RLHF / DPO / GRPO** | Reinforcement Learning from Human / AI Feedback, Direct Preference Optimization, Group Relative Policy Optimization (DeepSeek R1). |
| **Constitutional AI** | Anthropic's approach to scalable alignment via written principles. |
| **Mechanistic Interpretability** | Reverse-engineering neural networks — circuits, features, SAEs (Sparse Autoencoders). |
| **Red-Teaming / Adversarial Robustness** | Jailbreak research, prompt injection, model evaluations. |
| **Scalable Oversight** | Debate, recursive reward modeling, weak-to-strong generalization. |
| **AI Governance** | EU AI Act, US Executive Order 14110, Bletchley/Seoul declarations, frontier model risk frameworks. |
| **Catastrophic Risk** | Bio/chem/cyber/AI-R&D risk evals (METR, AISI UK, AISI US). |
