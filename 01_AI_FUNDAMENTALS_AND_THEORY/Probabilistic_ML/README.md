# Probabilistic ML

This directory covers probabilistic machine learning: uncertainty-aware models, Bayesian methods, latent-variable models, graphical models, and approximate inference.

## Content Map

| Subdirectory | Scope |
|---|---|
| `Gaussian_Processes/` | Kernel-based Bayesian regression/classification and uncertainty estimates. |
| `Hidden_Markov_Models/` | Sequential latent-state models for speech, biology, time series, and behavior modeling. |
| `MCMC_and_Sampling/` | Monte Carlo, MCMC, HMC, Gibbs sampling, and posterior simulation. |
| `Mixture_Models_and_EM/` | Gaussian mixtures, latent clusters, expectation-maximization, and soft assignment. |
| `Probabilistic_Graphical_Models/` | Bayesian networks, Markov random fields, factor graphs, and conditional independence. |
| `Probabilistic_Programming/` | Pyro, NumPyro, Stan, Turing, Bean Machine, and model-first inference workflows. |
| `Variational_Inference/` | Approximate posteriors, ELBO, amortized inference, and scalable Bayesian learning. |

## Enrichment Standard

Each page should state the random variables, likelihood, prior assumptions, inference method, uncertainty output, and evaluation metric.
