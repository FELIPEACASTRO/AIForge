# Model Merging

> Completely absent across the ENTIRE repo (grep for 'model merging', 'mergekit', 'model soup', 'task arithmetic' returns zero, including docs/). Model merging is now a mainstream, training-free way to combine fine-tuned checkpoints — it powers a large fraction of top Open LLM Leaderboard models and is a standard production technique. A clear high-value omission.
