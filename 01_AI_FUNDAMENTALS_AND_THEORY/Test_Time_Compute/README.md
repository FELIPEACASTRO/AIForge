# Test Time Compute

> The single biggest 2024-2026 paradigm shift (o1/o3/DeepSeek-R1/Gemini-Thinking): spending more compute at inference to scale reasoning. The repo has ZERO dedicated coverage of best-of-N, beam/lookahead search, self-consistency-as-scaling, process reward models, verifier-guided search, or the compute-optimal scaling laws for inference. The only 'test-time' hits in the repo are unrelated test-time augmentation/domain adaptation (Kaggle CV). This is a core fundamentals gap, not just a frontier note.
