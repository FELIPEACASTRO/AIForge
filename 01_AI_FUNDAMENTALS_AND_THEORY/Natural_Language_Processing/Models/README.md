# NLP Models

This directory covers model families for language understanding, generation, translation, retrieval, sequence labeling, and semantic representation.

## What Belongs Here

- model families, official docs, paper links, checkpoints, licenses, and intended-use notes
- architecture, training data caveats, context or input limits, benchmark evidence, and deployment constraints
- comparison tables across quality, latency, memory, price, modality, and safety behavior

## Evidence To Track

- Source URL, publication or update date, license, owner, and access conditions.
- Dataset/model/prompt version, benchmark split, metric, baseline, and known limitations when applicable.
- Clear separation between primary evidence, reproduced results, and AIForge interpretation.
