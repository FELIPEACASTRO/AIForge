# Natural Language Processing

This directory covers NLP foundations across linguistics, text preprocessing, representation learning, parsing, classification, information extraction, retrieval, translation, summarization, dialogue, and evaluation.

## Content Map

| Subdirectory | Scope |
|---|---|
| `Models/` | NLP model families, embeddings, transformers, sequence models, task heads, and language understanding architectures. |

## Core Topics

- Tokenization, normalization, segmentation, lemmatization, stemming, and sentence splitting.
- Sparse and dense text representations, embeddings, topic models, and semantic similarity.
- Sequence labeling, named entity recognition, parsing, relation extraction, and coreference.
- Retrieval, reranking, summarization, translation, question answering, and dialogue.
- Evaluation across accuracy, robustness, hallucination, factuality, bias, toxicity, multilinguality, and domain shift.

## Source Families

- Hugging Face Transformers and Datasets.
- spaCy pipelines and model catalogs.
- NLTK book and corpus resources.
- Stanford CoreNLP APIs and linguistic annotation tools.
- Papers with Code, ACL Anthology, and benchmark suites for NLP tasks.
