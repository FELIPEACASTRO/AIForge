# Mathematics For ML

This directory covers the mathematical toolkit behind machine learning, optimization, deep learning, probabilistic modeling, and evaluation.

## Content Map

| Subdirectory | Scope |
|---|---|
| `Linear_Algebra/` | Vectors, matrices, decompositions, eigensystems, embeddings, and tensor operations. |
| `Calculus_and_Differentiation/` | Gradients, Jacobians, Hessians, automatic differentiation, and backpropagation. |
| `Probability_Theory/` | Random variables, distributions, expectation, Bayes rule, and uncertainty. |
| `Statistics_Fundamentals/` | Estimation, inference, confidence intervals, hypothesis tests, and sampling. |
| `Convex_Optimization/` | Convexity, duality, constrained optimization, proximal methods, and guarantees. |
| `Numerical_Methods/` | Stability, approximation, iterative solvers, floating-point issues, and simulation. |
| `Discrete_Math_and_Graph_Theory/` | Graphs, combinatorics, logic, search, and discrete structures for AI. |

## Routing Rules

- Put applied statistical methods in `../Statistical_Learning/`.
- Put neural training algorithms in `../Deep_Learning/` or `../Optimization_Algorithms/`.
- Put production numerical concerns in `../../04_MLOPS_AND_PRODUCTION_AI/`.
