# Graph Neural Network Models

This directory covers GNN model families and architectures.

## Scope

- GCN, GraphSAGE, GAT, MPNN, graph transformers, heterogeneous GNNs, temporal GNNs, molecular GNNs, recommender GNNs, and knowledge-graph models.
- Track graph type, node/edge features, sampling strategy, task, scalability, inductive/transductive setting, and evaluation split.

## Reference Links

- PyTorch Geometric: https://pytorch-geometric.readthedocs.io/
- Deep Graph Library: https://www.dgl.ai/
- NVIDIA GNN frameworks: https://developer.nvidia.com/gnn-frameworks
- Papers with Code graph neural networks: https://paperswithcode.com/methods/category/graph-neural-networks

## Routing Rules

- Put general GNN theory in the parent directory.
- Put domain-specific graph applications in their vertical directories.
