# Graph Neural Networks

This directory covers graph representation learning, message passing, graph transformers, dynamic graphs, knowledge graphs, molecular graphs, recommender graphs, and network science for AI.

## Content Map

| Subdirectory | Scope |
|---|---|
| `Models/` | GCN, GAT, GraphSAGE, GIN, graph transformers, hypergraph models, and specialized architectures. |
| `dynamic_graphs/` | Temporal graph networks, evolving graphs, event streams, and temporal relational learning. |

## Application Areas

Molecules, social networks, recommender systems, fraud, cybersecurity, traffic, knowledge graphs, supply chains, and biomedical networks.

## Enrichment Standard

Record graph type, nodes, edges, features, task, split strategy, inductive/transductive setting, and metric.
