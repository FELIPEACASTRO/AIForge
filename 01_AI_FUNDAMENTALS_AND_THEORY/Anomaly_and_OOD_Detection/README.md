# Anomaly and OOD Detection

> Anomaly detection and OOD detection are core to reliable ML (fraud, monitoring, safety), but 'anomaly detection' surfaces only in GNN/agent-security/collection files — never as a topic. Deep SVDD, isolation forests, and modern OOD detectors have no home. High-value foundational and production-relevant gap.
