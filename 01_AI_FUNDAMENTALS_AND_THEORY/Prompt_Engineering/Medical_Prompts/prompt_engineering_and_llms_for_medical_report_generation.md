# Prompt Engineering and LLMs for Medical Report Generation

## Description

Comprehensive research on the latest prompt engineering techniques and Large Language Models (LLMs) applied to medical report generation (2023-2025). The focus is on prompting methodologies (Zero-shot, CoT, GKP) and state-of-the-art models such as MRG-LLM and Med-PaLM 2, with details on architecture, performance metrics, and practical prompt examples.

## Statistics

**MRG-LLM (2025):** Achieved state-of-the-art performance on chest X-ray datasets (IU X-ray and MIMIC-CXR), with the *Prompt-wise* model reaching BLEU-4 of 0.170 and ROUGE-L of 0.332 on IU X-ray. **Med-PaLM 2 (2023-2025):** Scores up to 86.5% on the MedQA dataset, improving by more than 19% over the original Med-PaLM. The multimodal version (Med-PaLM M) demonstrated an improvement of more than 8% in chest X-ray report generation scores.

## Features

Clinical Prompting Techniques (Zero-shot, Few-shot, Chain-of-Thought, Generated Knowledge, Meta-prompting); Specialized Multimodal LLM (MRG-LLM) for Radiology; High-Performance LLM (Med-PaLM 2) for Medical Questions and Report Generation.

## Use Cases

Automation of imaging report writing (Radiology); Clinical decision support and differential diagnosis; Summarization of medical history and treatment plans; Generation of educational materials for patients.

## Integration

**Prompt Example for Report Writing (Zero-shot):** "Act as an experienced physician and write a complete medical report for patient [Patient Name], based on the following data: [Insert patient data, history, physical examination findings, test results, and diagnosis]." **CoT Prompt Example:** "List the differential diagnoses for chest pain, specifying the step-by-step reasoning and the tests required for each one."
