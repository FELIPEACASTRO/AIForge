# Creative Prompts

This directory covers prompts for creative workflows: writing, design, games, worldbuilding, advertising, music, video, storyboarding, visual ideation, and concept exploration.

## Content Map

| Subdirectory | Scope |
|---|---|
| `Game_Design/` | Mechanics, levels, quests, character systems, balancing, narrative arcs, item design, and playtest-analysis prompts. |

## Prompt Patterns

- Style and constraint prompts for repeatable creative direction.
- Brief-to-concept prompts for ideation.
- Critique prompts for revision loops.
- Multimodal prompts that pair text with images, audio, video, or reference boards.
- Safety-aware prompts for rights, likeness, provenance, and brand constraints.

## Source Families

- OpenAI and Anthropic prompt-engineering docs.
- Google prompting guidance for Gemini.
- Public prompt libraries where license and authorship are clear.
- Creative tool documentation for image, audio, video, and multimodal models.
