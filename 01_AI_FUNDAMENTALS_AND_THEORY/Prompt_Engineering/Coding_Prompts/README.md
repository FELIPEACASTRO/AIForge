# Coding Prompts

This directory covers prompts for software engineering, code generation, debugging, testing, refactoring, documentation, code review, migration, and developer workflows.

## Scope

- Prompt patterns for repository inspection, issue reproduction, test generation, patch design, security review, API use, and migration planning.
- Evaluation criteria for correctness, minimal diffs, style consistency, dependency safety, and test evidence.
- Model/tool boundaries for file editing, terminal use, secrets, and production systems.
