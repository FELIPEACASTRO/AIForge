# Data Science Prompts

This directory covers prompts for data analysis, statistics, machine learning, experiment design, visualization, and data storytelling.

## Scope

- Prompts for EDA, hypothesis generation, data-cleaning plans, leakage checks, feature ideas, baseline design, error analysis, visualization, and reproducible reporting.
- Require dataset context, column meanings, target definition, split strategy, constraints, and output format.
