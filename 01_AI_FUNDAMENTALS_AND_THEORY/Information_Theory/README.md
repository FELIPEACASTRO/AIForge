# Information Theory

> Information theory is a load-bearing foundation for ML: cross-entropy/KL losses, the information bottleneck, mutual-information estimation (MINE/InfoNCE), rate-distortion, and MDL all derive from it. A grep over the entire repo finds zero substantive coverage (only incidental mentions in Kaggle/quantization docs). For a 'most complete' fundamentals index this is a glaring foundational hole.
