# Explainable AI

This directory covers interpretability, explainability, transparency, and human-understandable evidence for AI/ML systems.

## Scope

- Global and local explanations, feature attribution, counterfactuals, surrogate models, saliency, concept activation, inherently interpretable models, and explanation evaluation.
- Track audience, decision context, model type, explanation method, stability, faithfulness, and user actionability.
