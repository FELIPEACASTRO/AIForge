# Classical ML Algorithms

This directory covers non-deep-learning algorithms and statistical learning methods that remain essential for tabular data, small-data settings, interpretable modeling, feature engineering, and strong baselines.

## Content Map

| Subarea | Examples |
|---|---|
| Linear and generalized linear models | Linear regression, logistic regression, ridge, lasso, elastic net, GLMs. |
| Tree and ensemble methods | Decision trees, random forests, gradient boosting, XGBoost, LightGBM, CatBoost, bagging, stacking. |
| Kernel methods | SVMs, kernel ridge regression, Gaussian processes, kernel PCA. |
| Instance-based methods | kNN, nearest-centroid classifiers, metric learning baselines. |
| Clustering | k-means, hierarchical clustering, DBSCAN, HDBSCAN, Gaussian mixtures, spectral clustering. |
| Dimensionality reduction | PCA, ICA, NMF, t-SNE, UMAP, factor analysis. |
| Anomaly detection | Isolation forest, one-class SVM, LOF, robust covariance, statistical control charts. |

## Evaluation Notes

- Always pair algorithm pages with assumptions, failure modes, data requirements, interpretability notes, and baseline metrics.
- Prefer reproducible datasets from OpenML, UCI, Kaggle, and scikit-learn examples.
- Record whether a method is robust for tabular, sparse, time-series, text, image, or graph data.
