# Federated Learning

This directory covers learning from decentralized data without centralizing raw records.

## Scope

- Cross-device, cross-silo, secure aggregation, differential privacy, personalization, non-IID data, communication efficiency, and federated evaluation.
- Track client population, data heterogeneity, privacy assumptions, aggregation method, threat model, and evaluation protocol.

## Reference Links

- Flower framework: https://flower.ai/docs/
- TensorFlow Federated: https://www.tensorflow.org/federated
- FedML: https://doc.fedml.ai/
- OpenFL: https://openfl.readthedocs.io/

## Routing Rules

- Put privacy controls in the privacy/security directory.
- Put healthcare federated applications under healthcare AI if domain-specific.
