# Tabular Deep Learning

> Tabular data is the most common real-world ML modality, yet the pillar has no section on it. 'tabular' appears only in collection link-dumps and a Sparse-Transformer doc. The gradient-boosting-vs-deep-nets debate, FT-Transformer, TabPFN, and TabNet are high-value foundational topics with no canonical home.
