# AI Fundamentals And Theory

Core concepts, algorithms, math, and evaluation foundations behind artificial intelligence and machine learning. This pillar explains how AI systems work before they become model catalogs, datasets, production stacks, or vertical applications.

## Major Areas

| Area | What belongs here |
|---|---|
| Machine learning foundations | Supervised, unsupervised, semi-supervised, online, active, transfer, meta, few-shot, and continual learning. |
| Classical ML algorithms | Linear models, trees, ensembles, kernels, clustering, dimensionality reduction, anomaly detection, and evaluation. |
| Deep learning | Neural networks, CNNs, RNNs, transformers, attention, optimization, regularization, representation learning, and training strategies. |
| Generative models | GANs, VAEs, diffusion, flow models, EBMs, autoregressive models, and hybrid generative systems. |
| Retrieval and language foundations | NLP, information retrieval, RAG, long-context models, knowledge graphs, prompt engineering, and agentic reasoning. |
| Safety, privacy, and trust | Alignment, robustness, uncertainty, privacy, security, explainability, bias, and model evaluation. |
| Mathematical foundations | Linear algebra, calculus, probability, statistics, information theory, optimization, optimal transport, and graph theory. |

## Source Discipline

- Prefer textbooks, peer-reviewed papers, official framework docs, benchmark papers, and reproducible code.
- Put model-specific details in `../02_LLM_AND_AI_MODELS/`.
- Put datasets, prompt libraries, and source catalogs in `../03_DATASETS_TOOLS_AND_RESOURCES/`.
- Put deployment, monitoring, serving, and CI/CD content in `../04_MLOPS_AND_PRODUCTION_AI/`.
