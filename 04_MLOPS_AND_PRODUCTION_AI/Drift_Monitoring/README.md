# Drift Monitoring

> Drift detection is currently scattered as features inside vendor observability files (Evidently, Fiddler, WhyLabs, Vertex) but there is no dedicated topic consolidating the discipline — covariate/concept/label drift, statistical tests (PSI, KS, KL), and open-source drift libraries. A standalone section with methods + tools would strengthen the monitoring story beyond per-vendor pages.
