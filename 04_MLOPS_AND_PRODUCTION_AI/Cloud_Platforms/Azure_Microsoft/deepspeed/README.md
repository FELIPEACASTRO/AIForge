# Azure DeepSpeed

This directory covers DeepSpeed usage in Azure and Microsoft-oriented AI training workflows.

## Scope

- Distributed training, ZeRO optimization, memory savings, mixed precision, multi-node GPU jobs, and large-model training on Azure infrastructure.
- Track hardware, node count, GPU type, DeepSpeed config, framework, checkpointing, throughput, and failure recovery.
