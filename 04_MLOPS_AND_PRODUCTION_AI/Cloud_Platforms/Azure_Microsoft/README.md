# Azure and Microsoft AI Cloud

This directory covers Azure and Microsoft cloud resources for AI, ML, data, model serving, and MLOps.

## Scope

- Azure Machine Learning, Azure OpenAI, model registry, managed online endpoints, distributed training, data services, security, monitoring, and governance.
- Track service, region, quota, identity, network, data residency, cost, and deployment model.
