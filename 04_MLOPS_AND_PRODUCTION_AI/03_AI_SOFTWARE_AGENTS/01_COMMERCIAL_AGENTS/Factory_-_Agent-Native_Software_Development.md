# Factory | Agent-Native Software Development

## Descrição
Factory é uma plataforma de desenvolvimento de software nativa de agentes que permite delegar tarefas completas, como refatorações, resposta a incidentes e migrações, a "Droids" (agentes de IA) diretamente do IDE ou CI/CD. O foco é a segurança de nível empresarial e a integração com ferramentas de engenharia existentes.
