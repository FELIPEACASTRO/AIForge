# Devika - Agentic AI Software Engineer (Migrando para Opcode)

## Descrição
Engenheiro de software de IA agêntico que pode entender instruções humanas de alto nível, dividi-las em etapas, pesquisar informações relevantes e escrever código para atingir o objetivo. É um projeto de código aberto que visa ser uma alternativa ao Devin da Cognition AI. O projeto está sendo migrado para Opcode.
