# OpenHands: Code Less, Make More

## Descrição
Plataforma de agentes de desenvolvimento de software baseados em IA, que podem modificar código, executar comandos, navegar na web e interagir com APIs, atuando como um desenvolvedor humano. Anteriormente conhecido como OpenDevin.
