# AutoDev: The AI-powered coding wizard

## Descrição
Assistente de codificação e desenvolvimento de software baseado em IA para IntelliJ IDEA, com suporte multilíngue, geração automática de código, correção de bugs, prompts personalizáveis e recursos de agente autônomo para desenvolvimento, teste e documentação.
