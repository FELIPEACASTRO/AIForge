# codel - Agente de IA Autônomo para Desenvolvimento de Software

## Descrição
Agente de IA totalmente autônomo que pode realizar tarefas e projetos complexos usando terminal, navegador e editor. O agente é seguro, roda em um ambiente Docker isolado e possui um editor de texto e navegador integrados.
