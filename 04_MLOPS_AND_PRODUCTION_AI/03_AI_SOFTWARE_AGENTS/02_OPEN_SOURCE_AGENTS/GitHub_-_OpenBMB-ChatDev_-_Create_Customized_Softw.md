# GitHub - OpenBMB/ChatDev: Create Customized Software using Natural Language Idea (through LLM-powered Multi-Agent Collaboration)

## Descrição
ChatDev é uma empresa de software virtual que opera através de vários agentes inteligentes com diferentes papéis (CEO, CPO, CTO, Programador, Revisor, Testador, Designer de Arte). O objetivo é criar software personalizado usando uma ideia em linguagem natural através da colaboração multi-agente alimentada por LLM.
