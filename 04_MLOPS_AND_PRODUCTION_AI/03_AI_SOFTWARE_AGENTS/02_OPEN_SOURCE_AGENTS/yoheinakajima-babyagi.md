# yoheinakajima/babyagi

## Descrição
Framework experimental para um agente autônomo de auto-construção. O BabyAGI utiliza um novo framework de funções (functionz) para armazenar, gerenciar e executar funções a partir de um banco de dados, com o objetivo de desenvolver agentes autônomos que podem se aprimorar e expandir com complexidade mínima.
