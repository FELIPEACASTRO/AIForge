## Categories Covered

| Category | What's tracked |
|---|---|
| **Tracing** | LLM/tool/agent call spans, latency, cost, tokens |
| **Evals** | LLM-as-judge, programmatic checks, regression tests |
| **Prompt Management** | Versioning, A/B tests, rollouts |
| **Monitoring** | Drift, hallucination, toxicity, latency SLOs |
| **Datasets** | Golden sets, synthetic datasets, dataset versioning |
| **Annotations** | Human review, labeling, feedback loops |
