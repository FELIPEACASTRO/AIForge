# Edge and On Device Deployment

> Edge/on-device deployment as a production discipline (mobile/embedded MLOps, TinyML, OTA model updates, hardware accelerators, fleet management) has no dedicated section — edge appears only inside inference/quantization files (LLM_Inference, ONNX_Runtime).
