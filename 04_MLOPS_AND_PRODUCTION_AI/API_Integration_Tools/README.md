# API Integration Tools

This directory covers tools and patterns for integrating AI systems with APIs, services, data sources, and external tools.

## Scope

- API clients, tool calling, authentication, retries, rate limits, schemas, webhooks, observability, and safety boundaries.
- Track endpoint, auth model, data sensitivity, tool permissions, timeout, idempotency, and failure handling.

## Reference Links

- OpenAPI Specification: https://spec.openapis.org/oas/latest.html
- OpenAI function calling and tools: https://developers.openai.com/api/docs/guides/function-calling
- Model Context Protocol: https://modelcontextprotocol.io/
- FastAPI: https://fastapi.tiangolo.com/

## Routing Rules

- Put agent tool-use patterns in AI agents.
- Put production monitoring in observability directories.
