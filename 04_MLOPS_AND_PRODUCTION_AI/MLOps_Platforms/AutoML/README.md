# MLOps AutoML Platforms

This directory covers AutoML platforms and their production-management concerns.

## Scope

- Experiment search, feature search, model selection, pipeline generation, leaderboards, model export, governance, monitoring, and deployment handoff.
- Track search space, budget, validation scheme, explainability, reproducibility, and final artifact.
