# MLOps Observability

This directory covers observability for AI, ML, LLM, and agent systems.

## Scope

- Metrics, traces, logs, data drift, model drift, prompt traces, latency, cost, quality, safety, and incident response.
- Track signal, owner, threshold, alert, dashboard, retention, and remediation workflow.

## Reference Links

- OpenTelemetry: https://opentelemetry.io/docs/
- Evidently: https://docs.evidentlyai.com/
- MLflow tracing: https://mlflow.org/docs/latest/llms/tracing/
- Langfuse: https://langfuse.com/docs

## Routing Rules

- Put experiment tracking in `../Experiment_Tracking/`.
- Put model registry metadata in model registry directories.
