# MLOps Infrastructure

This directory covers infrastructure foundations for AI/ML platforms.

## Scope

- Compute, GPUs, Kubernetes, storage, networking, IAM, secrets, registries, CI/CD, observability, and cost controls.
- Track environment, region, cluster, hardware, quota, image, security boundary, and recovery plan.

- OpenTelemetry: https://opentelemetry.io/docs/
