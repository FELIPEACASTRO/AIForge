# MLOps Platforms

This directory covers platform-level MLOps systems and tools for managing model development, deployment, monitoring, governance, and collaboration.

## Scope

- Experiment tracking, model registry, feature management, workflow orchestration, serving, monitoring, CI/CD, lineage, governance, and access control.
- Platform comparisons by integration surface, deployment model, storage, observability, security, workflow support, and cost.
