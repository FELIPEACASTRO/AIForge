# Experiment Tracking

This directory covers experiment tracking for machine learning, LLMs, agents, and AI workflows.

## Scope

- Runs, parameters, metrics, artifacts, traces, datasets, prompts, model versions, and reproducibility metadata.
- Track code version, data version, environment, random seed, hardware, metrics, artifacts, and owner.

## Reference Links

- MLflow tracking: https://mlflow.org/docs/latest/ml/tracking/
- Weights & Biases: https://docs.wandb.ai/
- Neptune: https://docs.neptune.ai/
- Comet ML: https://www.comet.com/docs/

## Routing Rules

- Put model promotion in model-registry directories.
- Put observability of live systems in observability directories.
