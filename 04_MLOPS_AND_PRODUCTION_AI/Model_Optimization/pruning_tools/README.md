# Pruning Tools

This directory covers pruning tools and methods for model compression.

## Scope

- Structured pruning, unstructured pruning, channel pruning, sparsity, lottery-ticket methods, sensitivity analysis, and accuracy recovery.
- Track pruning method, target hardware, sparsity level, metric regression, latency impact, and deployment runtime support.

## Reference Links

- PyTorch pruning tutorial: https://docs.pytorch.org/tutorials/intermediate/pruning_tutorial.html
- TensorFlow Model Optimization: https://www.tensorflow.org/model_optimization
- ONNX Runtime performance: https://onnxruntime.ai/docs/performance/
- NNI pruning quickstart: https://nni.readthedocs.io/en/stable/tutorials/pruning_quick_start.html

## Routing Rules

- Put quantization and distillation in the broader model optimization directory.
- Put edge healthcare compression in healthcare edge directories.
