# Commercial AI Agents

This directory covers hosted, enterprise, and vendor-provided agent platforms and products.

## Scope

- Agent builders, copilots, enterprise automation agents, CRM/service agents, coding agents, data agents, and workflow agents.
- Product capabilities, integration model, security controls, audit logs, pricing model, deployment options, admin controls, and data-retention terms.
- Evaluation notes that separate marketing claims from tested behavior.

## Routing Rules

- Put open-source frameworks in `../02_OPEN_SOURCE_AGENTS/`.
- Put local agent system design in the parent `../` directory.
- Put vendor model details under the relevant model-provider folder when the model is the primary subject.
