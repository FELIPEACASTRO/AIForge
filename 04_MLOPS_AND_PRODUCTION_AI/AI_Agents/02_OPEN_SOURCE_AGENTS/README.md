# Open Source AI Agents

This directory covers open-source agent frameworks, libraries, runtimes, and reference applications.

## Scope

- Tool use, planning, memory, workflow graphs, multi-agent systems, human-in-the-loop controls, evaluation, tracing, and deployment.
- Framework comparison by license, language, model compatibility, state model, observability, tool permissions, and production readiness.
- Reproducible examples with pinned versions, test tasks, and eval traces.
