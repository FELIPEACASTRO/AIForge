# CropSight-US: An Object-based Crop Type Ground Truth Dataset

## Description

Conjunto de dados de verdade terrestre de tipo de cultura em escala continental e baseado em objetos para os Estados Unidos contíguos (CONUS). O dataset integra rótulos de cultura derivados de imagens do Google Street View com limites de campo delineados a partir de imagens Sentinel-2. É o primeiro dataset de verdade terrestre de tipo de cultura em escala nacional e baseado em objetos.

## Statistics

Cobre os anos de 2013 a 2023. Inclui mais de 100.000 objetos de verdade terrestre de tipo de cultura em 17 culturas principais e 294 Distritos de Estatísticas Agrícolas. Cada objeto é acompanhado por uma pontuação de incerteza.

## Features

Dados de satélite Sentinel-2 (para delineamento de limites de campo) e imagens do Google Street View (para rotulagem). Dados de verdade terrestre baseados em objetos (com limites de campo explícitos), o que permite a extração de informações de textura e estrutura em nível de campo. Inclui pontuação de incerteza para cada rótulo.
