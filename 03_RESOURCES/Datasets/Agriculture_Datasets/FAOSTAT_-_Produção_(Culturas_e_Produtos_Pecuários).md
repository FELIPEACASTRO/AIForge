# FAOSTAT - Produção (Culturas e Produtos Pecuários)

## Description

O FAOSTAT é o portal de dados estatísticos da Organização das Nações Unidas para a Alimentação e a Agricultura (FAO). O domínio "Produção - Culturas e Produtos Pecuários" fornece dados abrangentes sobre a produção agrícola global, incluindo área colhida, rendimento e quantidade de produção para mais de 278 produtos.

## Statistics

Dados anuais de 1961 até o ano mais recente (2023, com atualizações em 2025). Abrange 199 países e territórios. Mais de 278 produtos.

## Features

Dados de nível nacional e regional. Inclui culturas primárias (cereais, frutas, fibras, etc.) e produtos pecuários. Permite download em massa (Bulk Downloads) em formato .zip.
