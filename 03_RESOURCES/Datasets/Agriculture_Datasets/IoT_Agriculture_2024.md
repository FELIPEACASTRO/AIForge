# IoT Agriculture 2024

## Description

O conjunto de dados foi coletado a partir de uma estufa inteligente equipada com tecnologias avançadas para monitoramento e controle de condições ambientais, como parte de uma pesquisa de mestrado (2023-2024) na Universidade de Tikrit, Iraque. Inclui um aplicativo que conecta os dados ao Google Sheets para monitoramento e controle remoto, fornecendo uma plataforma eficaz para o gerenciamento eficiente da estufa.

## Statistics

13 features, 37923 linhas (Rows). Atualização esperada: Semanal.

## Features

date (datetime64), temperature (int64), humidity (int64), water_level (int64), N (int64 - Nível de Nitrogênio), P (int64 - Nível de Fósforo), K (int64 - Nível de Potássio), Fan_actuator_OFF/ON (float64), Watering_plant_pump_OFF/ON (float64), Water_pump_actuator_OFF/ON (float64).

## Use Cases

Monitoramento e controle remoto de estufas, análise de condições ambientais para otimização de crescimento de plantas, desenvolvimento de modelos de classificação e previsão em agricultura de precisão.

## Integration

Disponível para download direto no Kaggle. Pode ser importado e analisado usando bibliotecas padrão de ciência de dados como Pandas em Python. O estudo original utilizou o Google Sheets para monitoramento e controle remoto.
