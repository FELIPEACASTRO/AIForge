# DDInter 2.0

## Description

Base de dados abrangente, profissional e de acesso aberto que fornece informações extensas sobre interações medicamento-medicamento (DDIs), medicamento-alimento (DFIs), interações de similaridade medicamento-doença (DDSIs) e duplicações terapêuticas. Esta versão aprimorada oferece dados expandidos, com anotações detalhadas como descrições de mecanismos, níveis de risco, estratégias de gerenciamento e códigos ATC atualizados.

## Statistics

302.516 associações DDI, conectando 2.290 medicamentos aprovados. Também inclui DFIs e DDSIs. Última atualização: 14 de maio de 2024.

## Features

Interações medicamento-medicamento, medicamento-alimento, medicamento-doença, duplicações terapêuticas. Anotações detalhadas: mecanismo, nível de risco, estratégias de gerenciamento, códigos ATC.

## Use Cases

Suporte à decisão clínica, pesquisa em segurança de medicamentos, verificação de prescrições, estudos de farmacovigilância.

## Integration

Acesso via interface web para pesquisa e verificação de interações. Não há API pública ou download direto de dataset mencionado na página principal.
