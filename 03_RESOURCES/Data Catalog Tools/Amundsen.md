# Amundsen

## Description

Amundsen é um motor de descoberta de dados e metadados de código aberto, originalmente desenvolvido pelo Lyft. Seu objetivo é melhorar a produtividade de analistas, cientistas e engenheiros de dados ao interagir com dados. Ele funciona indexando recursos de dados (tabelas, dashboards, streams, etc.) e utilizando um algoritmo de busca estilo PageRank baseado em padrões de uso. É frequentemente descrito como o "Google Search para dados".

## Statistics

Estrelas no GitHub: ~4.7k. Licença: Apache 2.0. Hospedado pela LF AI & Data Foundation.

## Features

Interface de Usuário intuitiva; Suporte a Entidades (Tabelas, Dashboards, Streams, Pessoas); Estatísticas de Colunas de Tabela; Orquestração ETL; Arquitetura de Microsserviços (Frontend, Metadata, Search); Databuilder para ingestão de dados.

## Use Cases

Melhorar a produtividade de analistas, cientistas e engenheiros de dados; Descoberta de dados baseada em padrões de uso (PageRank); Indexação de diversos recursos de dados.

## Integration

Integrações Suportadas com diversos conectores de Tabela e Dashboard. Arquitetura de microsserviços permite integração via APIs. O Databuilder é a biblioteca de ingestão de dados.
