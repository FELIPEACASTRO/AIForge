# Medical Imaging Transfer Learning

This directory covers transfer learning for medical imaging models.

## Scope

- Pretrained vision models, medical foundation models, fine-tuning, linear probing, adapter methods, self-supervised pretraining, and domain adaptation.
- Track source dataset, target modality, frozen layers, adaptation data, metric, site shift, scanner shift, and clinical review.

## Reference Links

- MONAI documentation: https://monai-dev.readthedocs.io/
- PyTorch transfer learning tutorial: https://docs.pytorch.org/tutorials/beginner/transfer_learning_tutorial.html
- TorchVision models: https://docs.pytorch.org/vision/main/models.html
- timm documentation: https://huggingface.co/docs/timm/index

## Routing Rules

- Put general transfer learning in the healthcare transfer-learning parent directory.
- Put segmentation-specific transfer notes in medical image segmentation.
