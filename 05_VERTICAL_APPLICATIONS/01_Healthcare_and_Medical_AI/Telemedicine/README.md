# Telemedicine AI

This directory covers AI for telemedicine, remote care, virtual triage, patient engagement, remote monitoring, and digital front-door workflows.

## Scope

- Symptom intake, visit summarization, remote monitoring alerts, care navigation, virtual assistants, patient education, and follow-up automation.
- Track patient safety, escalation policy, consent, accessibility, privacy, multilingual support, and clinical review.

## Reference Links

- WHO digital health: https://www.who.int/health-topics/digital-health
