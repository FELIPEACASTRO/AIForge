# Drug Discovery AI Models

This directory covers model families used in AI-assisted drug discovery.

## Scope

- Molecular property prediction, docking, generative chemistry, retrosynthesis, protein-ligand interaction, ADMET, toxicity, and active-learning models.
- Track molecular representation, target assay, dataset split, experimental validation, uncertainty, and applicability domain.

## Reference Links

- DeepChem: https://deepchem.io/
- Therapeutics Data Commons: https://tdcommons.ai/
- ChEMBL: https://www.ebi.ac.uk/chembl/
- PubChem: https://pubchem.ncbi.nlm.nih.gov/
- AlphaFold Protein Structure Database: https://alphafold.ebi.ac.uk/

## Routing Rules

- Put docking-specific resources in `../../Molecular_Docking/`.
- Put protein structure models in `../../Protein_Structure_Prediction/`.
- Put papers in healthcare paper or model paper directories.
