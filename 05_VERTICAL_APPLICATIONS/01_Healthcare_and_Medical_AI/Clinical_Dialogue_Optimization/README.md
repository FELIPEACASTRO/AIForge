# Clinical Dialogue Optimization

This directory covers AI for improving clinical conversations, documentation, patient communication, and care-team dialogue.

## Scope

- Visit summarization, patient question answering, care-plan explanation, clinician-patient communication, call-center triage, and dialogue quality review.
- Track safety, consent, PHI handling, escalation, citation, clinician review, and patient comprehension.
