# Protein Structure Prediction

This directory covers AI systems for protein structure prediction, complex prediction, folding, embeddings, structure search, and biological interpretation.

## Scope

- AlphaFold-style models, ESMFold, RoseTTAFold, OpenFold, protein language models, structure databases, and downstream biomedical use.
- Track sequence source, model version, confidence metrics, template/MSA use, limitations, licensing, and experimental validation status.

## Reference Links

- AlphaFold Protein Structure Database: https://alphafold.ebi.ac.uk/
- RCSB Protein Data Bank: https://www.rcsb.org/
- UniProt: https://www.uniprot.org/
- ESM repository: https://github.com/facebookresearch/esm
- OpenFold: https://github.com/aqlaboratory/openfold
- EMBL-EBI AlphaFold training: https://www.ebi.ac.uk/training/online/courses/alphafold/
