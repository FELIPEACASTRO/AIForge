# Medical Imaging Transfer Learning

This directory covers transfer learning specifically inside medical imaging workflows.

## Scope

- Pretraining transfer, domain adaptation, few-shot learning, self-supervised transfer, and foundation-model adaptation.
- Track source dataset, target dataset, modality, site shift, scanner shift, label schema, metric, and calibration.
