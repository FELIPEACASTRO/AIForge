# Medical Imaging Generative Models

This directory covers generative models for medical imaging.

## Scope

- Diffusion models, GANs, VAEs, image synthesis, reconstruction, modality translation, segmentation assistance, and augmentation.
- Track modality, generation purpose, privacy risk, artifact risk, clinical validity, and downstream impact.
