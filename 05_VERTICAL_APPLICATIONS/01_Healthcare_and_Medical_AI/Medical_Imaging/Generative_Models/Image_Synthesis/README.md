# Medical Image Synthesis

This directory covers generative AI for medical image synthesis, augmentation, reconstruction, translation, and simulation.

## Scope

- Diffusion, GANs, VAEs, image-to-image translation, synthetic lesions, modality translation, de-identification support, and data augmentation.
- Track privacy, realism, clinical utility, artifact risk, label validity, downstream impact, and regulatory limitations.

- MONAI Generative Models: https://github.com/Project-MONAI/GenerativeModels

- Put general generative medical imaging in the parent generative-model directory.
- Put synthetic EHR or tabular data in healthcare synthetic-data generation.
