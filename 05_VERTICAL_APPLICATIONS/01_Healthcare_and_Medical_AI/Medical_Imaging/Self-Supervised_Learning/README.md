# Medical Imaging Self-Supervised Learning

This directory covers self-supervised and weakly supervised learning for medical imaging.

## Scope

- Contrastive learning, masked image modeling, reconstruction pretexts, multimodal image-report pretraining, and label-efficient adaptation.
- Track pretraining data, leakage risk, downstream task, evaluation split, and clinical failure modes.
