# Medical Segmentation Foundation Models

This directory covers foundation models and promptable systems for medical image segmentation.

## Scope

- SAM-style medical adaptation, generalist segmentation models, organ/lesion/cell segmentation, promptable segmentation, and foundation-model fine-tuning.
- Track modality, anatomical coverage, prompt type, training data, license, metric, and failure modes.

## Reference Links

- MONAI documentation: https://monai-dev.readthedocs.io/
- Medical Segmentation Decathlon: http://medicaldecathlon.com/
- nnU-Net: https://github.com/MIC-DKFZ/nnUNet
- Segment Anything Model: https://github.com/facebookresearch/segment-anything

## Routing Rules

- Put general segmentation in the parent directory.
- Put model compression for edge deployment in healthcare edge AI.
