# Medical Imaging Fine-Tuning

This directory covers fine-tuning workflows for medical imaging models.

## Scope

- Full fine-tuning, partial fine-tuning, adapters, LoRA, linear probes, prompt tuning, and low-data adaptation.
- Track pretrained model, target dataset, labels, augmentation, split, metric, calibration, and failure cases.

- MONAI documentation: https://monai-dev.readthedocs.io/
- PyTorch transfer learning tutorial: https://docs.pytorch.org/tutorials/beginner/transfer_learning_tutorial.html
