# Medical Imaging Frameworks

This directory covers frameworks and toolkits for medical imaging AI.

## Scope

- Training frameworks, preprocessing libraries, DICOM workflows, segmentation frameworks, deployment SDKs, and evaluation tooling.
- Track modality support, license, file formats, model families, deployment support, and reproducibility.

## Reference Links

- MONAI documentation: https://monai-dev.readthedocs.io/
- nnU-Net: https://github.com/MIC-DKFZ/nnUNet
- SimpleITK: https://simpleitk.org/
- pydicom: https://pydicom.github.io/
- MONAI Deploy App SDK: https://github.com/Project-MONAI/monai-deploy-app-sdk

## Routing Rules

- Put model-specific resources in the relevant medical imaging model directory.
- Put deployment resources in healthcare edge or MLOps.
