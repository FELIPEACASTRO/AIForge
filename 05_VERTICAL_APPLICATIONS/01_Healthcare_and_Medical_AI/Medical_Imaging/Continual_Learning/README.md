# Medical Imaging Continual Learning

This directory covers continual, lifelong, and adaptive learning for medical imaging models.

## Scope

- Continual updates across hospitals, scanners, protocols, time, patient cohorts, and labels.
- Track replay policy, privacy constraints, drift detection, forgetting, validation data, and clinical release controls.

## Reference Links

- MONAI documentation: https://monai-dev.readthedocs.io/
- Papers with Code continual learning: https://paperswithcode.com/task/continual-learning
- The Cancer Imaging Archive: https://www.cancerimagingarchive.net/
