# Source-Free Domain Adaptation for Medical Imaging

This directory covers source-free domain adaptation and continual adaptation methods for medical imaging.

## Scope

- Adaptation without source data, test-time adaptation, privacy-preserving adaptation, hospital/scanner/domain shift, and continual updating.
- Track target data access, source model assumptions, privacy constraints, validation protocol, and catastrophic forgetting.

## Reference Links

- MONAI documentation: https://monai-dev.readthedocs.io/
- Papers with Code domain adaptation: https://paperswithcode.com/task/domain-adaptation
- The Cancer Imaging Archive: https://www.cancerimagingarchive.net/

## Routing Rules

- Put general continual learning in the parent continual-learning directory.
- Put medical imaging transfer learning in sibling transfer-learning directories.
