# Proteomics AI

This directory covers AI for proteomics, protein quantification, mass spectrometry, protein function, biomarker discovery, and multi-omics integration.

## Scope

- Proteomics data processing, peptide identification, protein inference, biomarker prediction, pathway analysis, and multi-omics modeling.
- Track assay, instrument, preprocessing, normalization, batch effects, missingness, and biological validation.
