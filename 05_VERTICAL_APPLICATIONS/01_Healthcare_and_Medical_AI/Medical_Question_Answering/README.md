# Medical Question Answering

This directory covers medical QA systems, benchmarks, prompts, and model evaluations.

## Scope

- Clinical QA, consumer health QA, biomedical literature QA, exam QA, retrieval-grounded QA, and citation-based answering.
- Track source corpus, model, retrieval method, answer evidence, safety boundary, and clinical review.

## Reference Links

- PubMed: https://pubmed.ncbi.nlm.nih.gov/
- MedQA dataset: https://github.com/jind11/MedQA
- Open Medical-LLM Leaderboard: https://huggingface.co/blog/leaderboard-medicalllm
- FDA AI/ML in SaMD: https://www.fda.gov/medical-devices/software-medical-device-samd/artificial-intelligence-software-medical-device

## Routing Rules

- Put medical LLM model catalogs in `../Medical_LLMs/`.
- Put medical prompts in prompt-engineering medical prompts.
