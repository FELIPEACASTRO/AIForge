# Clinical NLP

This directory covers NLP for clinical text, biomedical text, EHR notes, radiology reports, discharge summaries, trial matching, coding, phenotyping, and patient communication.

## Scope

- Named entity recognition, relation extraction, summarization, classification, cohort discovery, de-identification, coding, and RAG over clinical documents.
- Medical terminology, ontologies, abbreviations, temporal reasoning, negation, uncertainty, and privacy constraints.
- Evaluation with clinician review, PHI protection, calibration, subgroup analysis, and factuality checks.
