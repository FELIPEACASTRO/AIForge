# Molecular Docking AI

This directory covers AI-assisted molecular docking, binding prediction, virtual screening, and protein-ligand modeling.

## Scope

- Docking scores, pose prediction, binding affinity prediction, generative design loops, active learning, and structure-aware screening.
- Track target structure, ligand library, pocket definition, docking engine, benchmark, experimental validation, and uncertainty.
