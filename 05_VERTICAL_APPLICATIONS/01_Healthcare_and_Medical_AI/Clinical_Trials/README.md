# Clinical Trials AI

This directory covers AI for clinical trials, trial design, recruitment, matching, monitoring, synthetic controls, endpoint analysis, and real-world evidence.

## Scope

- Eligibility extraction, cohort discovery, patient-trial matching, site selection, protocol optimization, adverse-event monitoring, and trial document analysis.
- Track inclusion/exclusion criteria, data source, consent, bias, regulatory constraints, and validation evidence.
