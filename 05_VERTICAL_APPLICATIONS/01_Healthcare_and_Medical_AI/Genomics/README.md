# Genomics AI

This directory covers AI for genomics, precision medicine, variant calling, gene expression, multi-omics, population genetics, and biomedical discovery.

## Content Map

| Area | Scope |
|---|---|
| Variant analysis | Variant calling, pathogenicity prediction, annotation, prioritization, and rare disease workflows. |
| Multi-omics | Genomics, transcriptomics, epigenomics, proteomics, metabolomics, and integrated patient representations. |
| Foundation models | DNA/RNA/protein language models, sequence transformers, and biological embeddings. |
| Clinical genomics | Precision medicine, pharmacogenomics, cancer genomics, inherited disease, and clinical reporting. |
| Data governance | Consent, ancestry bias, privacy, controlled-access datasets, and reproducibility. |

## Enrichment Standard

Include species, assay, cohort, reference genome, preprocessing pipeline, split strategy, benchmark, and ethical constraints.
