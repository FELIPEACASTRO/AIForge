# Genomics Protein Folding

This directory covers links between genomic variation and protein folding, structure, and functional interpretation.

## Scope

- Variant effect prediction, protein structure impact, missense interpretation, pathogenicity, and structure-guided functional annotation.
- Track variant, transcript, protein isoform, structure source, confidence metric, and clinical evidence.

## Reference Links

- AlphaFold Protein Structure Database: https://alphafold.ebi.ac.uk/
- UniProt: https://www.uniprot.org/
- ClinVar: https://www.ncbi.nlm.nih.gov/clinvar/
- Ensembl Variant Effect Predictor: https://www.ensembl.org/info/docs/tools/vep/index.html
- RCSB Protein Data Bank: https://www.rcsb.org/

## Routing Rules

- Put general protein folding in `../../../Protein_Structure_Prediction/`.
- Put variant-calling pipelines in the parent directory.
