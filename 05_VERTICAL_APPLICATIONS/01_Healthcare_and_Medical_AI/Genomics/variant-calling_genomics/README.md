# Variant Calling Genomics

This directory covers AI and statistical methods for variant calling and genomic analysis.

## Scope

- SNPs, indels, structural variants, germline/somatic calling, long-read sequencing, short-read sequencing, quality control, annotation, and clinical interpretation.
- Track reference genome, sequencing platform, coverage, truth set, sample type, benchmark, and regulatory or clinical context.

## Reference Links

- Genome in a Bottle: https://www.nist.gov/programs-projects/genome-bottle
- GATK best practices: https://gatk.broadinstitute.org/hc/en-us
- NCBI ClinVar: https://www.ncbi.nlm.nih.gov/clinvar/
- gnomAD: https://gnomad.broadinstitute.org/
- DeepVariant: https://github.com/google/deepvariant
