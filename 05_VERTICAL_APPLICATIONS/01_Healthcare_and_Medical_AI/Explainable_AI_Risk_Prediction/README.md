# Explainable Healthcare Risk Prediction

This directory covers explainable risk-prediction systems in healthcare.

## Scope

- Clinical risk scores, deterioration risk, readmission risk, mortality risk, disease progression, adverse-event risk, and treatment-risk stratification.
- Track target timing, cohort, calibration, subgroup performance, explanation method, clinician actionability, and safety.
