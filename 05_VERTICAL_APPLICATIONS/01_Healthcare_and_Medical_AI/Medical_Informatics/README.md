# Medical Informatics

This directory covers healthcare data standards, clinical data models, EHR interoperability, coding systems, and informatics workflows for AI.

## Scope

- FHIR, OMOP, SNOMED CT, LOINC, ICD, RxNorm, EHR data pipelines, clinical data harmonization, and data-quality checks.
- Record schema, vocabulary version, provenance, PHI handling, cohort logic, and governance requirements.

## Reference Links

- HL7 FHIR: https://hl7.org/fhir/
- OHDSI OMOP Common Data Model: https://www.ohdsi.org/data-standardization/
- MIMIC-IV: https://physionet.org/content/mimiciv/
- OHDSI MIMIC OMOP conversion: https://github.com/OHDSI/MIMIC
