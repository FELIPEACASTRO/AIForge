# Healthcare Synthetic Data Generation

This directory covers synthetic healthcare data for AI development, evaluation, privacy-preserving analysis, simulation, and regulatory science.

## Scope

- Synthetic EHR, imaging, claims, genomics, trial, patient journey, device, and population-health data.
- Track generation method, privacy claim, utility metric, bias risks, validation method, and whether synthetic data may be used for training, testing, or documentation only.
