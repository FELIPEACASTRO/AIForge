# Epidemiological Surveillance

This directory covers AI-assisted surveillance for public-health signals and outbreaks.

## Scope

- Syndromic surveillance, notifiable disease tracking, wastewater signals, search/social signals, lab reports, case reports, and early warning systems.
- Track reporting lag, geographic granularity, case definition, denominator, false alarms, and public-health action.

## Reference Links

- WHO data: https://data.who.int/
- CDC data: https://data.cdc.gov/
- CDC wastewater surveillance: https://www.cdc.gov/nwss/
- Our World in Data: https://ourworldindata.org/
