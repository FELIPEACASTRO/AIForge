# Medical Image Segmentation

This directory covers medical image segmentation resources that are not yet routed into the larger medical imaging segmentation tree.

## Scope

- Organ, lesion, tumor, vessel, tissue, cell, and anatomy segmentation.
- Track modality, annotation source, label schema, metric, inter-rater variability, and clinical use case.

## Reference Links

- MONAI documentation: https://monai-dev.readthedocs.io/
- nnU-Net: https://github.com/MIC-DKFZ/nnUNet
- Medical Segmentation Decathlon: http://medicaldecathlon.com/
- Grand Challenge: https://grand-challenge.org/

## Routing Rules

- Prefer the main `../Medical_Imaging/Segmentation/` tree for new segmentation material.
- Put edge compression in healthcare edge compression directories.
