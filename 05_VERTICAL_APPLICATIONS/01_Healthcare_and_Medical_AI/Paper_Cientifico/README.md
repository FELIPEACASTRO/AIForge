# Healthcare Scientific Papers

This directory covers scientific papers and reading notes for healthcare AI.

## Scope

- Clinical AI papers, biomedical papers, medical imaging papers, medical LLM papers, validation studies, regulatory papers, and review articles.
- Record evidence type, cohort, method, claim, limitation, code/data availability, and clinical relevance.

## Reference Links

- PubMed: https://pubmed.ncbi.nlm.nih.gov/
- Europe PMC: https://europepmc.org/
- medRxiv: https://www.medrxiv.org/
- arXiv: https://arxiv.org/
- Papers with Code medical tasks: https://paperswithcode.com/area/medical

## Routing Rules

- Put model cards in model directories.
- Put datasets in dataset directories.
- Put paper-derived project implementation in project-showcase directories.
