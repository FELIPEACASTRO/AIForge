# Clinical Decision Support and Biomedical NLP

This directory covers overlap between clinical decision support and biomedical NLP.

## Scope

- Evidence retrieval, guideline matching, phenotype extraction, note summarization, risk explanation, and clinical recommendations grounded in biomedical text.
- Track source evidence, clinical guideline, NLP model, uncertainty, actionability, and clinician review.

## Reference Links

- FDA clinical decision support software guidance: https://www.fda.gov/regulatory-information/search-fda-guidance-documents/clinical-decision-support-software
