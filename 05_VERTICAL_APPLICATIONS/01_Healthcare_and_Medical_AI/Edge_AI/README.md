# Healthcare Edge AI

This directory covers AI deployed near clinical devices, bedside systems, mobile workflows, hospital edge servers, and resource-constrained medical environments.

## Scope

- On-device inference, offline operation, privacy-preserving inference, low-latency alerts, imaging devices, wearables, ICU monitors, and remote care.
- Track hardware, model size, latency, calibration, failure handling, update path, cybersecurity, and clinical safety constraints.

## Reference Links

- LiteRT: https://developers.google.com/edge/litert
- ONNX Runtime: https://onnxruntime.ai/docs/
- MONAI Deploy App SDK: https://github.com/Project-MONAI/monai-deploy-app-sdk
- TensorRT: https://docs.nvidia.com/deeplearning/tensorrt/

## Routing Rules

- Put compression methods in `Model_Compression/`.
- Put radiology/imaging applications in medical imaging directories.
- Put platform deployment details in MLOps.
