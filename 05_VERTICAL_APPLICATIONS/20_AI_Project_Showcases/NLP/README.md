# NLP Projects

This directory covers applied NLP project examples: classification, extraction, retrieval, summarization, translation, dialogue, semantic search, RAG, and language-model evaluation.

## Project Types

- Text classification and moderation.
- Named entity recognition and information extraction.
- Document understanding and table extraction.
- Semantic search, reranking, and RAG.
- Summarization, translation, and multilingual systems.
- LLM evaluation, prompt testing, and hallucination analysis.

## Project Metadata

Each project should record data source, license, task definition, model family, tokenizer, metric, baseline, evaluation split, failure modes, safety concerns, and deployment status.

## Source Families

- Hugging Face Transformers, Datasets, Evaluate, spaCy, NLTK, Stanford CoreNLP, Haystack, LlamaIndex, LangChain, and retrieval benchmarks.
- ACL Anthology, Papers with Code, HELM, lm-evaluation-harness, RAGAS, Promptfoo, and task-specific benchmarks.
