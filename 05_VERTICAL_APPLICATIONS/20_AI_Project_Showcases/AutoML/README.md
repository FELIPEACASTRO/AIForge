# AutoML Projects

This directory covers project examples that use automated machine learning systems.

## Scope

- Tabular AutoML, time-series AutoML, neural architecture search, feature search, hyperparameter optimization, and model selection pipelines.
- Record framework, search space, time budget, metric, validation split, leaderboard risk, explainability, and final model export.

## Reference Links

- AutoGluon: https://auto.gluon.ai/
- H2O AutoML: https://docs.h2o.ai/h2o/latest-stable/h2o-docs/automl.html
- FLAML: https://microsoft.github.io/FLAML/
- AutoML benchmark frameworks: https://openml.github.io/automlbenchmark/frameworks.html
