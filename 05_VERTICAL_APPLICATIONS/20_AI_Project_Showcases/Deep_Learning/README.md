# Deep Learning Projects

This directory covers complete deep-learning project examples across vision, text, audio, video, graphs, time series, reinforcement learning, and scientific ML.

## Project Types

- CNN, RNN, transformer, diffusion, graph neural network, autoencoder, and reinforcement-learning projects.
- Transfer learning, self-supervised learning, fine-tuning, distillation, and deployment examples.
- Training loops, distributed training, mixed precision, checkpointing, and evaluation pipelines.

## Required Metadata

Each project should record dataset, architecture, loss, optimizer, schedule, augmentation, hardware, training time, metric, baseline, inference path, and failure modes.

## Source Families

- PyTorch, TensorFlow, JAX, Keras, Lightning, Hugging Face, timm, MONAI, TorchAudio, TorchVision, and Papers with Code.
- Benchmark suites and official tutorials for the task domain.
