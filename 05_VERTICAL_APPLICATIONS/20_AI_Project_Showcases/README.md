# AI Project Showcases

This directory collects practical AI project examples, implementation patterns, demos, notebooks, open-source projects, and applied workflows. It should help readers move from concepts to working systems.

## Project Families

| Subdirectory | Scope |
|---|---|
| `Applied_ML/` | End-to-end ML examples across common business and research tasks. |
| `Deep_Learning/` | Neural-network projects, training workflows, and reproducible architectures. |
| `Generative_AI/` | LLM, image, video, audio, and multimodal generation projects. |
| `MLOps_Projects/` | Production pipelines, model serving, monitoring, CI/CD, and deployment examples. |
| `ML_Frameworks/` | Framework-specific projects and examples. |
| `NLP/`, `Multimodal/`, `Audio_Generation/`, `Video_Generation/` | Modality-specific applications. |
| `Reinforcement_Learning/`, `Recommender_Systems/`, `Time_Series/`, `XAI/` | Specialized applied ML project categories. |
| `Kaggle/` | Competition notebooks, datasets, solution patterns, and score-oriented workflows. |

## Project Readiness Checklist

- Problem statement and target user.
- Dataset, license, and provenance.
- Model or method used.
- Training or inference path.
- Evaluation metric and baseline.
- Reproducibility instructions.
- Deployment or demo route.
- Known limitations, risks, and next improvements.
