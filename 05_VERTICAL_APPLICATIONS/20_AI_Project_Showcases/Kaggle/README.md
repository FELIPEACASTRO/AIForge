# Kaggle — Competitions, Winning Solutions & Playbook

The most complete map of **Kaggle competitive ML** in AIForge: a live index of competitions, the top public solution notebooks per competition (pulled from the Kaggle API), curated winning-solution write-ups by category, and the recurring techniques that win.

## Contents

| File / Folder | What's inside | Source |
|---|---|---|

## How to use this

1. **Browse competitions** → `Competitions_Index.md`.
2. **Find the best public code** for one → `Top_Solution_Notebooks.md` (or that competition's Discussion tab for gold write-ups).
3. **Learn the patterns** that win → `Winning_Solutions/Grandmaster_Playbook.md`, then the category file matching your problem.

> Notebook lists and competition metadata were pulled from the **public Kaggle API**. Curated write-ups were assembled from public sources; verify specific links before relying on them.

- [More_Competitive_ML_and_HuggingFace_Resources.md](./More_Competitive_ML_and_HuggingFace_Resources.md) — 46 additional verified resources (double-check).
