# Source Taxonomy And Methodology

This country coverage uses the 195 independent-country baseline already used by AIForge. It is not a claim that every country has an active Biohub Kaggle team or a national cell-tracking program. It is a search-routing matrix for the Biohub - Cell Tracking During Development topic and adjacent evidence: cell tracking, bioimage analysis, developmental microscopy, light-sheet microscopy, OME-Zarr/image data infrastructure, and microscopy communities.

## Evidence Tiers

| Tier | Meaning |
|---|---|
| S3 direct Biohub/Kaggle source | Direct official competition, host, or repository connection. |
| S2 country-specific source | A country-specific bioimaging, microscopy, image-analysis, imaging-infrastructure, or institutional source was found. |
| S1 regional source route | A credible regional network covers the country/region, but this pass did not capture a country-specific source. |
| S0 search backlog | No country-specific or regional source strong enough for this topic was captured in this pass. |

## What Counts As Related To Biohub Cell Tracking

- Direct sources: the Kaggle competition, Biohub, Royer Lab, official metric repository, and public competition notebooks.
- Core methods: cell detection, segmentation, lineage tracking, division detection, graph linking, Ultrack, Trackastra, Cellpose, StarDist, OME-Zarr, GEFF and tracksdata.
- Country routes: microscopy facilities, bioimaging networks, image-analysis communities, public image-data infrastructure, and developmental-biology imaging groups.
