# Kaggle Winning Solutions

This directory covers winning and high-performing Kaggle solution writeups, notebooks, code patterns, and reproducible lessons.

## Scope

- Solution summaries, competition context, data leakage notes, validation design, model families, feature engineering, ensembles, and post-processing.
- Keep public-score and private-score claims dated and tied to the original competition or official discussion.

## Reference Links

- Kaggle competitions: https://www.kaggle.com/competitions
- Kaggle datasets: https://www.kaggle.com/datasets
- Kaggle models: https://www.kaggle.com/models
- Kaggle notebooks: https://www.kaggle.com/code

## Routing Rules

- Put generic applied ML projects in sibling project-showcase directories.
- Put dataset catalogs in the dataset pillar.
- Put reusable feature engineering in the data-engineering or fundamentals tree.
