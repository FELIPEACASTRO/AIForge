# Applied ML Projects

This directory covers practical machine-learning projects that solve a defined problem with measurable outcomes.

## Project Types

- Classification, regression, ranking, recommendation, forecasting, anomaly detection, clustering, and decision-support systems.
- Tabular ML, time-series ML, geospatial ML, fraud detection, churn prediction, risk scoring, demand forecasting, and operational analytics.
- Hybrid systems that combine rules, ML models, human review, and business constraints.

## Required Metadata

Each project should record problem framing, decision owner, data source, leakage risks, split strategy, baseline, metric, cost of errors, model card, deployment status, and monitoring plan.

## Source Families

- scikit-learn, XGBoost, LightGBM, CatBoost, statsmodels, PyCaret, AutoGluon, MLflow, DVC, Evidently, and cloud reference architectures.
- Domain benchmarks and public datasets with clear licenses.
