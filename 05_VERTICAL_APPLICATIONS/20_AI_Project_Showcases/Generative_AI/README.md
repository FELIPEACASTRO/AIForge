# Generative AI Projects

This directory covers projects that generate text, images, audio, video, code, molecules, designs, simulations, plans, or multimodal outputs.

## Project Types

- LLM applications, RAG, agents, copilots, prompt apps, and structured-output systems.
- Image, video, music, speech, and multimodal generation workflows.
- Synthetic data, simulation, molecule generation, design generation, and creative tooling.
- Evaluation harnesses for quality, safety, factuality, bias, cost, latency, and user acceptance.

## Required Metadata

Record model, provider or checkpoint, prompt template, retrieval source, safety controls, output schema, evaluation set, human-review process, cost estimate, and failure cases.

## Source Families

- OpenAI, Anthropic, Google Gemini, Hugging Face, Stability AI, Replicate, Runway, Meta, Mistral, Cohere, NVIDIA, and open-source model hubs.
- Evaluation tools such as OpenAI Evals, Promptfoo, Ragas, DeepEval, lm-evaluation-harness, HELM, and human preference benchmarks.
