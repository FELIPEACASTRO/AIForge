# Science AI

This directory covers AI for scientific discovery, simulation, literature mining, experimental design, materials, biology, chemistry, physics, climate science, astronomy, and laboratory automation.

## Scope

- Scientific foundation models, surrogate models, inverse design, active learning, Bayesian optimization, and simulation acceleration.
- Protein, molecule, materials, climate, physics, astronomy, microscopy, and instrument-data applications.
- Reproducibility, uncertainty quantification, scientific validity, and dataset provenance.

## Reference Links

- AlphaFold Protein Structure Database: https://alphafold.ebi.ac.uk/
- Materials Project: https://next-gen.materialsproject.org/
- NASA Open Data: https://data.nasa.gov/
- arXiv: https://arxiv.org/
- Papers with Code: https://paperswithcode.com/
- Hugging Face papers: https://huggingface.co/papers

## Routing Rules

- Put protein structure work in `../01_Healthcare_and_Medical_AI/Protein_Structure_Prediction/`.
- Put scientific model catalogs in `../../02_LLM_AND_AI_MODELS/Scientific_Models/`.
