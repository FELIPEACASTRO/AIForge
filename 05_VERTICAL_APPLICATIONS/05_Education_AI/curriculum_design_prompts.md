# Curriculum Design Prompts

## Description
**Curriculum Design Prompts** are detailed and structured instructions provided to Large Language Models (LLMs) or other Artificial Intelligence (AI) tools to assist educators, instructional designers, and school administrators in the creation, adaptation, and optimization of educational materials and lesson plans. Rather than generic requests, these prompts are crafted to include pedagogical context, target audience (grade level, age), learning objectives, format constraints (duration, activity type), and assessment criteria. The effective use of these prompts turns AI into a powerful planning assistant, reducing administrative workload and allowing educators to focus on direct interaction with students [1] [2].

## Examples
```
1.  **Creating a Detailed Lesson Plan:**
    ```
    Act as an expert instructional designer. Create a 50-minute lesson plan for 9th-grade students on the "Industrial Revolution". The plan should include:
    1. Learning Objective (according to Bloom's taxonomy).
    2. A 5-minute "hook" activity.
    3. Main content divided into 3 sections with varied teaching methods (discussion, visual, reading).
    4. A 15-minute hands-on activity that promotes critical thinking.
    5. A formative assessment exit question (exit ticket).
    ```

2.  **Differentiation and Support for ELL/Special Needs Students:**
    ```
    Rewrite the following text about "Food Chains" for a Lexile 600-800 reading level. Additionally, generate a list of 5 key terms with simplified definitions and translations into Portuguese and Spanish, aimed at supporting English as a Second Language (ELL) students.
    [Insert the original text here]
    ```

3.  **Generating Standards-Aligned Assessment Questions:**
    ```
    Create 5 intermediate-level multiple-choice questions (C3 - Application) on the topic "Newton's Laws". The questions should be aligned with the Curriculum Standard [Insert the standard code here, e.g., BNCC EM13CHS101]. Include the correct answer and a rationale for each distractor.
    ```

4.  **Developing an Assessment Rubric:**
    ```
    Design a 4-level analytic rubric (Beginner, Developing, Proficient, Exceeding) to assess a high school research project on "Urban Sustainability". The rubric should focus on three main criteria: Depth of Research, Organization and Clarity, and Innovative Solution Proposals.
    ```

5.  **Creating an Interactive Activity:**
    ```
    Generate the script for a 20-minute role-playing game for a 7th-grade History class on "Ancient Greece". The game should have 4 main roles (Philosopher, Soldier, Merchant, Citizen) and a learning objective focused on understanding the social structure of the polis.
    ```

6.  **Creating Content for Online Modules (Distance Learning):**
    ```
    Act as an expert in instructional design for distance learning. Create the outline of a 2-hour module on "Introduction to Python Programming". The outline should include: Module Title, 4 Sections with estimated duration, 3 learning objectives per section, and a suggestion for a practical assessment at the end of the module.
    ```
```

## Best Practices
*   **Be Specific and Contextualized:** Include the AI's role (e.g., "Act as an instructional designer"), the target audience (grade level, age, special needs), and the desired output format (40-minute plan, analytic rubric, 5 questions) [1].
*   **Define Constraints and Standards:** Explicitly mention curriculum standards (e.g., BNCC, Common Core), the taxonomy of objectives (e.g., Bloom), or the reading level (e.g., Lexile) to ensure pedagogical alignment [1] [3].
*   **Use the Iterative Approach:** Start with a broad prompt and refine the result with follow-up prompts (e.g., "Adapt this plan to include a 10-minute activity for students with dyslexia") [2].
*   **Focus on Quality, Not Quantity:** Use AI to generate high-quality drafts that the educator can review and personalize, rather than trying to generate the complete final product in a single prompt.
*   **Integrate Metacognition:** Ask the AI to justify its pedagogical choices (e.g., "Explain why this activity is effective for developing critical thinking") [1].

## Use Cases
*   **Lesson and Unit Planning:** Rapid generation of lesson plan drafts, teaching sequences, and learning objectives [1].
*   **Differentiation and Inclusion:** Adapting materials for different reading levels, languages (ELL), or special educational needs (IEP) [1].
*   **Assessment Creation:** Developing multiple-choice questions, essay questions, rubrics, and standards-aligned assessment criteria [3].
*   **Professional Development:** Creating training modules for teachers on new methodologies or technologies.
*   **Stakeholder Communication:** Generating newsletters for parents or progress reports aligned with performance data [2].

## Pitfalls
*   **Generic Prompts:** Vague requests (e.g., "Create a lesson plan") result in superficial and unusable content, requiring a lot of rework [1].
*   **Lack of Pedagogical Alignment:** Failing to specify curriculum standards or learning objectives can lead to materials misaligned with the official curriculum.
*   **Over-reliance:** Treating the AI's output as a final product without critical review can introduce factual errors, biases, or inappropriate pedagogical choices [2].
*   **Bias and Inclusion:** AI can perpetuate cultural or gender biases if the prompt does not explicitly require consideration of diversity and inclusion.
*   **Ignoring Local Context:** AI does not know the specific reality of the classroom, the available resources, or the school culture, which requires manual personalization [3].
