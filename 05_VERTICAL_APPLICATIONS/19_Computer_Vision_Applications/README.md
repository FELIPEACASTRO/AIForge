# Computer Vision Applications

This directory covers applied computer vision systems across detection, classification, segmentation, tracking, OCR, inspection, remote sensing, medical imaging, retail, robotics, and safety.

## Scope

- Image and video models, datasets, labeling, augmentation, evaluation, deployment, and human review.
- Track modality, annotation protocol, model family, metrics, error slices, latency, safety, and deployment environment.

## Reference Links

- OpenCV: https://opencv.org/
- TorchVision models: https://docs.pytorch.org/vision/main/models.html
- Hugging Face computer vision course: https://huggingface.co/learn/computer-vision-course/en/unit0/welcome/welcome
- COCO dataset: https://cocodataset.org/

- Put vision model catalogs in `../../02_LLM_AND_AI_MODELS/Vision_Models/`.
