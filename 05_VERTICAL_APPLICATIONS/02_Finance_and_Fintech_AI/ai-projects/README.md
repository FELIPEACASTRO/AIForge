# FinTech AI Projects

This directory covers finance and fintech AI project examples that do not fit a narrower finance subdirectory yet.

## Scope

- Proofs of concept, notebooks, reference architectures, datasets, evaluation notes, and production lessons for financial AI.
- Track problem, data source, model, validation split, leakage risk, metric, compliance risk, and deployment status.

## Reference Links

- FINRA AI in securities: https://www.finra.org/rules-guidance/key-topics/fintech/report/artificial-intelligence-in-the-securities-industry
- CFPB AI in financial services comment: https://www.consumerfinance.gov/about-us/newsroom/cfpb-comment-on-request-for-information-on-uses-opportunities-and-risks-of-artificial-intelligence-in-the-financial-services-sector/
- FRED: https://fred.stlouisfed.org/
- Kaggle datasets: https://www.kaggle.com/datasets

## Routing Rules

- Move fraud, credit, risk, trading, and payment projects into their dedicated subdirectories when the domain is clear.
