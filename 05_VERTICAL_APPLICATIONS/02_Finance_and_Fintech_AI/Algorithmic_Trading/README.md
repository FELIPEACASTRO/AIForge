# Algorithmic Trading

This directory covers AI/ML for trading research, signal generation, execution, market microstructure, portfolio construction, and trading-system risk controls.

## Scope

- Forecasting, classification, reinforcement learning, alternative data, feature engineering, backtesting, and execution algorithms.
- Market data quality, survivorship bias, look-ahead leakage, transaction costs, slippage, latency, capacity, and regime shift.
- Compliance, supervision, market manipulation controls, kill switches, and model-risk management.
