# FinTech AI GitHub Repositories

This directory covers GitHub repositories related to finance and fintech AI.

## Scope

- Open-source examples, libraries, datasets, notebooks, risk tools, fraud examples, credit-scoring examples, and market-analysis projects.
- Record repository owner, license, activity, language, data source, model type, and reproducibility status.
