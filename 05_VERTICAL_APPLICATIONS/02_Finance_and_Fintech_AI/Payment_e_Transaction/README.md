# Payment and Transaction AI

This directory covers AI for payment systems, transaction analytics, authorization, routing, reconciliation, risk scoring, dispute handling, and operational monitoring.

## Scope

- Card, ACH, instant payment, bank transfer, wallet, merchant, cross-border, and open-banking transaction workflows.
- Transaction embeddings, anomaly detection, fraud adjacency, payment routing, chargeback prediction, merchant categorization, and customer-support automation.
- Compliance with privacy, security, model-risk, audit, and consumer-protection expectations.
