# FinTech AI Tutorials

This directory covers tutorial material for finance and fintech AI.

## Scope

- Step-by-step examples for fraud, credit scoring, risk, time series, customer analytics, payments, and market data.
- Tutorials should include dataset source, license, task, metric, baseline, leakage check, and limitations.

## Reference Links

- scikit-learn user guide: https://scikit-learn.org/stable/user_guide.html
- FRED: https://fred.stlouisfed.org/
- Kaggle datasets: https://www.kaggle.com/datasets
- FINRA AI report: https://www.finra.org/rules-guidance/key-topics/fintech/report/artificial-intelligence-in-the-securities-industry

## Routing Rules

- Move mature tutorials into the relevant finance subdirectory.
- Put production risk/governance notes in risk management.
