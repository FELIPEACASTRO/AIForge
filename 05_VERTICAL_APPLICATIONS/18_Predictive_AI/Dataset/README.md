# Predictive AI Datasets

This directory covers datasets used for predictive AI examples and benchmarkable predictive workflows.

## Scope

- Classification, regression, forecasting, ranking, anomaly detection, churn, risk, demand, maintenance, fraud, and operational prediction datasets.
- Dataset cards should include target definition, unit of observation, leakage risks, split strategy, time coverage, license, missingness, and baseline metric.

## Reference Links

- OpenML: https://www.openml.org/
- UCI Machine Learning Repository: https://archive.ics.uci.edu/
- Kaggle datasets: https://www.kaggle.com/datasets
- Data.gov: https://data.gov/
- scikit-learn toy datasets: https://scikit-learn.org/stable/datasets.html

## Routing Rules

- Put framework guidance in sibling `../Framework/`.
- Put official guide content in `../Guia_Oficial/`.
- Put domain-specific datasets in the relevant dataset modality or vertical folder.
