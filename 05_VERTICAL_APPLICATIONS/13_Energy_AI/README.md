# Energy AI

AI for **grid management**, **renewable forecasting**, **energy storage**, **fusion control**, **smart buildings**, and **resource exploration**.

## Grid & Power

- **Open Power System Data** — https://open-power-system-data.org/
- **Pecan Street Dataport** — https://www.pecanstreet.org/dataport/
- **PJM Interconnection Open Data** — https://dataminer2.pjm.com/
- **GraphCast / GenCast (DeepMind)** — weather + grid forecasting
- **NeuralProphet / Darts / NeuralForecast** — load forecasting

## Renewable Forecasting

- **OpenClimateFix** — https://openclimatefix.org/
- **PV-LIB / pvlib-python** — https://pvlib-python.readthedocs.io/
- **Wind Toolkit (NREL)** — https://www.nrel.gov/grid/wind-toolkit.html
- **Skyfora / Climavision** — commercial weather AI
