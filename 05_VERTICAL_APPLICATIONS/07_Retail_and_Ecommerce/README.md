# Retail and Ecommerce AI

This directory covers AI for retail, ecommerce, marketplaces, merchandising, search, personalization, pricing, inventory, and customer experience.

## Scope

- Recommendation, search ranking, demand forecasting, fraud, churn, dynamic pricing, catalog enrichment, visual search, customer service, and logistics.
- Track data source, customer privacy, metric, experimentation plan, bias, and operational decision.

## Reference Links

- RecBole recommender library: https://recbole.io/
- TensorFlow Recommenders: https://www.tensorflow.org/recommenders
- Amazon Personalize: https://docs.aws.amazon.com/personalize/
- Google Cloud retail recommendations: https://docs.cloud.google.com/retail/docs/predict

## Routing Rules

- Put recommender project examples in project showcases.
- Put time-series demand forecasting in time-series project folders.
