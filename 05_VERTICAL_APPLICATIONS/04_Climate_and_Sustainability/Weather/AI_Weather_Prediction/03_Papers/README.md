# Weather AI Papers

Research papers and paper-discovery entry points for AI weather prediction, graph neural weather models, foundation models for forecasting, nowcasting, reanalysis, climate emulation, and uncertainty quantification.

## Starting Points

- [GraphCast](https://www.science.org/doi/10.1126/science.adi2336): graph neural network weather forecasting.
- [Pangu-Weather](https://www.nature.com/articles/s41586-023-06185-3): 3D deep learning for global weather forecasting.
- [FourCastNet](https://arxiv.org/abs/2202.11214): Fourier neural operator weather forecasting.
- [WeatherBench](https://arxiv.org/abs/2002.00469): benchmark for data-driven medium-range weather prediction.
