# Weather AI

This directory covers AI and ML for weather forecasting, nowcasting, data assimilation support, severe-weather detection, reanalysis emulation, and forecast post-processing.

## Scope

- Numerical weather prediction augmentation, graph neural weather models, diffusion/transformer weather models, radar nowcasting, ensemble calibration, and extreme-event forecasting.
- Reanalysis, forecast, radar, satellite, station, ocean, and climate data sources.
- Verification with CRPS, RMSE, anomaly correlation, Brier score, reliability, lead-time skill, and regional/event slicing.
