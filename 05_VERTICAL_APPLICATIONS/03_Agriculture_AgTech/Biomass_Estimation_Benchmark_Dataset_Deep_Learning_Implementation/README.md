# Biomass Estimation Benchmark and Implementation

This directory covers benchmark datasets and deep-learning implementations for biomass estimation.

## Scope

- Remote sensing, drone imagery, field measurements, LiDAR, canopy height, vegetation indices, and deep-learning baselines.
- Track target unit, sensor, geography, split, metric, baseline, and reproducibility.

## Reference Links

- NASA Earthdata: https://www.earthdata.nasa.gov/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- Radiant MLHub on AWS Open Data: https://registry.opendata.aws/radiant-mlhub/
- PyTorch tutorials: https://pytorch.org/tutorials/

## Routing Rules

- Put raw datasets in AgTech biomass dataset folders.
- Put papers in AgTech biomass paper folders.
