# Crop Growth Prediction

This directory covers AI models that predict crop growth over time.

## Scope

- Phenology, biomass accumulation, canopy growth, growth stage forecasting, yield precursor modeling, and climate/management response.
- Track time horizon, crop, geography, weather, soil, management inputs, and temporal validation.

## Reference Links

- USDA NASS: https://www.nass.usda.gov/
- NASA Earthdata: https://www.earthdata.nasa.gov/
- Copernicus Climate Data Store: https://cds.climate.copernicus.eu/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
