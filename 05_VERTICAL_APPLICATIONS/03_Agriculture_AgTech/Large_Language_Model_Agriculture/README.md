# Agriculture Large Language Models

This directory covers LLMs and language-enabled AI systems for agriculture.

## Scope

- Agronomic assistants, crop advisory, extension support, pest/disease QA, farm operations, multimodal field reports, and knowledge retrieval.
- Track source corpus, language, region, crop, retrieval sources, safety, and expert review.

## Reference Links

- FAO data: https://www.fao.org/faostat/
- CGIAR: https://www.cgiar.org/
- PubAg: https://pubag.nal.usda.gov/
- Hugging Face models: https://huggingface.co/models

## Routing Rules

- Put model catalogs in LLM/model directories.
- Put crop disease work in plant disease and crop health folders.
