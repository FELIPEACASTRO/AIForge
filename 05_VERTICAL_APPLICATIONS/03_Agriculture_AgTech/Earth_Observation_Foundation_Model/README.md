# Earth Observation Foundation Models

This directory covers foundation models for Earth observation and agriculture.

## Scope

- Satellite foundation models, geospatial embeddings, remote-sensing transformers, crop monitoring, land cover, biomass, and climate/agriculture transfer.
- Track sensor, bands, resolution, geography, pretraining data, downstream task, and license.

## Reference Links

- Prithvi foundation model: https://huggingface.co/ibm-nasa-geospatial/Prithvi-100M
- NASA Earthdata: https://www.earthdata.nasa.gov/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- Copernicus Data Space: https://dataspace.copernicus.eu/

## Routing Rules

- Put crop-specific applications in crop subdirectories.
- Put general climate/geospatial datasets in the datasets pillar.
