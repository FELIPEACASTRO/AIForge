# Biomass Estimation from Remote Sensing

This directory covers biomass estimation using satellite, aerial, drone, SAR, LiDAR, and other remote-sensing data.

## Scope

- Above-ground biomass, crop biomass, canopy height, vegetation structure, phenology, and multi-sensor fusion.
- Track sensor, bands, spatial resolution, temporal window, target unit, ground truth, and uncertainty.

## Reference Links

- NASA Earthdata: https://www.earthdata.nasa.gov/
- GEDI data: https://gedi.umd.edu/data/products/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- Copernicus Data Space: https://dataspace.copernicus.eu/

## Routing Rules

- Put generic biomass datasets in AgTech biomass dataset folders.
- Put vegetation index features in feature-engineering directories.
