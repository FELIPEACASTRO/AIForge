# Crop Health Monitoring

This directory covers AI for monitoring crop health using remote sensing, field imagery, weather, soil, and sensor data.

## Scope

- Stress detection, disease risk, nutrient deficiency, drought impact, canopy condition, and temporal monitoring.
- Track crop, sensor, date, geography, index/features, labels, and agronomic action.

## Reference Links

- NASA Earthdata: https://www.earthdata.nasa.gov/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- USDA NASS: https://www.nass.usda.gov/
- Radiant MLHub on AWS Open Data: https://registry.opendata.aws/radiant-mlhub/

## Routing Rules

- Put disease-specific detection in plant disease or crop disease directories.
- Put vegetation-index features in feature-engineering directories.
