# Integrated Agricultural AI Framework

This directory covers integrated agricultural AI spanning yield prediction, pest and disease detection, AGI-style planning, and biomass estimation.

## Scope

- Multi-task crop systems, agronomic decision support, remote sensing, field imagery, weather, pest pressure, disease risk, biomass, and yield.
- Track crop, geography, tasks, model architecture, data sources, decision outputs, uncertainty, and validation across seasons.

## Routing Rules

- Split mature work into yield, disease, biomass, or monitoring subdirectories once the primary task is clear.
