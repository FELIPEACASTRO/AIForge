# TinyML Edge Deployment for Plant Disease AI

This directory covers edge and TinyML deployment for plant disease detection.

## Scope

- On-device inference, compressed models, microcontrollers, mobile devices, field cameras, LoRa/IoT, and low-power disease detection workflows.
- Track hardware, model size, latency, battery, connectivity, threshold, and field validation.

## Reference Links

- TensorFlow Lite Micro: https://www.tensorflow.org/lite/microcontrollers
- Edge Impulse docs: https://docs.edgeimpulse.com/
- LiteRT: https://developers.google.com/edge/litert
- PlantVillage dataset: https://github.com/spMohanty/PlantVillage-Dataset

## Routing Rules

- Put general plant disease detection in plant disease directories.
- Put model compression in MLOps optimization directories.
