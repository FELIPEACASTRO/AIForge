# Crop Growth Stage Recognition

This directory covers AI for recognizing crop growth stages and phenological phases.

## Scope

- Image, remote-sensing, time-series, field-sensor, and multimodal phenology recognition.
- Track crop, stage taxonomy, observation cadence, sensor, label protocol, and temporal validation.
