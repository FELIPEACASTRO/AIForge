# Few-Shot Learning for Rare Crop Diseases

This directory covers few-shot and low-data learning for rare plant diseases.

## Scope

- Few-shot classification, metric learning, transfer learning, synthetic augmentation, active learning, and field-validation protocols.
- Track crop, disease, label count, image conditions, domain shift, and agronomic risk.
