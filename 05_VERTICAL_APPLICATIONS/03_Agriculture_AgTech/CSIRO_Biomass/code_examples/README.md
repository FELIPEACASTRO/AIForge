# CSIRO Biomass Code Examples

This directory stores code examples for AgTech biomass modeling workflows.

## Scope

- Training, inference, losses, optimizers, preprocessing, and reusable code snippets.
- Examples should state input data assumptions, target variable, sensor modality, model family, metric, and reproducibility status.
- Keep runnable code separate from paper notes and dataset documentation.

## Reference Links

- PyTorch documentation: https://pytorch.org/docs/stable/index.html
- scikit-learn user guide: https://scikit-learn.org/stable/user_guide.html
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- NASA Earthdata: https://www.earthdata.nasa.gov/

## Routing Rules

- Put Python packages and modules in `src/`.
- Put biomass papers in `../AgTech_Biomass/Paper_ArXiv/`.
- Put dataset notes in `../AgTech_Biomass/Dataset/`.
