# CSIRO Biomass Documentation

This directory stores documentation for the CSIRO biomass subproject.

## Scope

- Method notes, data assumptions, model architecture, training/inference instructions, metrics, benchmarks, and reproducibility notes.
- Keep code in `../code_examples/` and keep papers/datasets in AgTech biomass topic folders.

## Reference Links

- NASA Earthdata: https://www.earthdata.nasa.gov/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- PyTorch tutorials: https://pytorch.org/tutorials/
- scikit-learn user guide: https://scikit-learn.org/stable/user_guide.html

## Routing Rules

- Put executable examples in code examples.
- Put source papers in paper directories.
