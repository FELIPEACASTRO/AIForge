# AgTech Synthetic Data Generation

This directory covers synthetic data generation for agriculture AI.

## Scope

- Synthetic crop images, disease samples, weed masks, remote-sensing simulations, weather scenarios, phenology trajectories, and farm-operation data.
- Track generation method, label validity, realism, bias, license, and downstream validation.

## Reference Links

- BlenderProc: https://dlr-rm.github.io/BlenderProc/
- NVIDIA Omniverse Replicator: https://docs.omniverse.nvidia.com/py/replicator/
- Albumentations: https://albumentations.ai/docs/
- PlantVillage dataset: https://github.com/spMohanty/PlantVillage-Dataset

## Routing Rules

- Put augmentation-only methods in data augmentation.
- Put disease-specific synthetic examples in plant disease folders.
