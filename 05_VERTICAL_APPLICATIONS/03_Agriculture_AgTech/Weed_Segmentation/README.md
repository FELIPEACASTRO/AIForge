# Weed Segmentation

This directory covers AI for segmenting weeds and crops in field imagery.

## Scope

- Semantic segmentation, instance segmentation, crop/weed separation, multispectral UAV imagery, edge deployment, and field validation.
- Track crop, weed classes, sensor, annotations, metric, weather/lighting, and deployment constraints.

## Reference Links

- WeedsGalore paper: https://arxiv.org/abs/2502.13103
- PlantVillage dataset: https://github.com/spMohanty/PlantVillage-Dataset
- Papers with Code segmentation: https://paperswithcode.com/task/semantic-segmentation
- TensorFlow Lite examples: https://www.tensorflow.org/lite/examples

## Routing Rules

- Put transfer learning for weed segmentation in `Transfer_Learning/`.
- Put general plant disease detection in plant disease folders.
