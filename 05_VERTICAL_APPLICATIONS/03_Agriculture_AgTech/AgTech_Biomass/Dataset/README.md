# AgTech Biomass Datasets

This directory covers datasets for biomass estimation, vegetation monitoring, crop phenotyping, canopy structure, and remote-sensing agriculture workflows.

## Scope

- Field biomass measurements, satellite/drone imagery, LiDAR, multispectral/hyperspectral data, weather, soil, crop type, and management variables.
- Record sensor, location, date, crop, target unit, sampling protocol, split design, license, and known measurement error.

## Reference Links

- Radiant MLHub on AWS Open Data: https://registry.opendata.aws/radiant-mlhub/
- NASA Earthdata: https://www.earthdata.nasa.gov/
- USDA NASS: https://www.nass.usda.gov/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- Copernicus Climate Data Store: https://cds.climate.copernicus.eu/
