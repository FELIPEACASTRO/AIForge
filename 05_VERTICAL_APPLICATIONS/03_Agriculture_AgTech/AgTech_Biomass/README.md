# AgTech Biomass

This directory covers biomass estimation and monitoring in agriculture using AI, remote sensing, geospatial features, field measurements, and crop models.

## Content Map

| Area | Scope |
|---|---|
| Data sources | Satellite imagery, drone imagery, field plots, LiDAR, weather, soil, and management records. |
| Feature engineering | Vegetation indices, texture, canopy height, phenology, topography, and climate covariates. |
| Models | Random forests, gradient boosting, CNNs, transformers, geospatial foundation models, and hybrid crop models. |
| Evaluation | Plot-level validation, spatial holdout, temporal holdout, RMSE/MAE/R2, and uncertainty. |

## Enrichment Standard

Record crop, biomass target definition, measurement protocol, spatial resolution, temporal window, and validation split.
