# AgTech Biomass Papers

This directory covers papers and research notes for biomass estimation, crop monitoring, remote sensing, phenotyping, and agricultural foundation models.

## Scope

- arXiv and peer-reviewed papers with data, methods, code, evaluation, sensors, geography, crop, and biomass target definition.
- Separate directly measured biomass from proxies such as vegetation indices, canopy height, yield, or phenology.

## Reference Links

- arXiv: https://arxiv.org/
- Semantic Scholar: https://www.semanticscholar.org/
- Papers with Code agriculture tasks: https://paperswithcode.com/area/computer-vision/agriculture
- NASA Earthdata: https://www.earthdata.nasa.gov/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets

## Routing Rules

- Put datasets in sibling `../Dataset/`.
- Put implementation code under the CSIRO biomass code-example tree.
- Put general agriculture datasets in `../../../../03_DATASETS_TOOLS_AND_RESOURCES/Datasets/Agriculture_Datasets/`.
