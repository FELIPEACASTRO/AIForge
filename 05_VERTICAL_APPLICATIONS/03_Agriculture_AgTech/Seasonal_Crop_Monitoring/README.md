# Seasonal Crop Monitoring

This directory covers AI for monitoring crop conditions across a growing season.

## Scope

- Phenology, growth-stage tracking, health monitoring, biomass, stress detection, weather response, and yield precursors.
- Track crop, season, geography, observation cadence, sensor, temporal split, and agronomic decision.

## Reference Links

- NASA Earthdata: https://www.earthdata.nasa.gov/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- Copernicus Climate Data Store: https://cds.climate.copernicus.eu/
- USDA NASS: https://www.nass.usda.gov/

## Routing Rules

- Put growth-stage recognition in crop growth-stage directories.
- Put crop health signals in crop health monitoring.
