# Plant Disease Detection

This directory covers AI for detecting plant diseases from images, field sensors, remote sensing, and multimodal farm data.

## Scope

- Classification, detection, segmentation, severity scoring, few-shot detection, edge deployment, and agronomic action support.
- Track crop, disease, image condition, label source, geography, season, metric, and field validation.

## Reference Links

- PlantVillage dataset: https://github.com/spMohanty/PlantVillage-Dataset
- FAO plant health: https://www.fao.org/plant-health-2020
- Papers with Code agriculture tasks: https://paperswithcode.com/area/computer-vision/agriculture
