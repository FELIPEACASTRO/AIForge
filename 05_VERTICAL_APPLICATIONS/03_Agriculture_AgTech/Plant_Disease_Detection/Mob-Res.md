# Mob-Res

## Description

A lightweight and explainable Convolutional Neural Network (CNN) model that combines residual learning (residual blocks) with the MobileNetV2 feature extractor. Designed for plant disease diagnosis, it is well suited for mobile applications due to its low number of parameters.

## Statistics

Parameters: 3.51 million. Average Accuracy: 97.73% (Plant Disease Expert, 58 classes). Accuracy: 99.47% (PlantVillage, 38 classes). Publication: Scientific Reports (2025).

## Features

Hybrid Architecture (residual blocks and MobileNetV2); Lightweight (3.51 million parameters); Explainability (Explainable AI).

## Use Cases

Fast and accurate diagnosis of plant diseases in the field; Mobile applications to assist farmers; Outbreak prevention and crop protection.

## Integration

Requirements: Python 3.10, TensorFlow 2.10, Keras 2.10, Numpy 1.26. Code and pre-trained model (.keras) available in the GitHub repository.
