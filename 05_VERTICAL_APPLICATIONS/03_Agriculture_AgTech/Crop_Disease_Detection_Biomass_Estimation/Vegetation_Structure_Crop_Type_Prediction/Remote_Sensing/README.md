# Remote Sensing for Vegetation Structure

This directory covers remote-sensing methods for vegetation structure and crop type prediction.

## Scope

- Satellite, drone, multispectral, hyperspectral, SAR, LiDAR, time-series, and phenology features.
- Track sensor, bands, resolution, acquisition dates, cloud masking, label source, and spatial validation.

## Reference Links

- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- NASA Earthdata: https://www.earthdata.nasa.gov/
- GEDI data: https://gedi.umd.edu/data/products/
- Copernicus Data Space: https://dataspace.copernicus.eu/

## Routing Rules

- Put vegetation index features in feature-engineering directories.
- Put biomass-specific resources in biomass estimation directories.
