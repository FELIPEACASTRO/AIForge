# Vegetation Structure and Crop Type Prediction

This directory covers AI for predicting vegetation structure and crop type in combined disease/biomass workflows.

## Scope

- Crop type classification, canopy structure, vegetation morphology, biomass proxies, remote sensing, and field imagery.
- Track sensor, crop taxonomy, spatial resolution, labels, metric, and seasonal validation.

## Reference Links

- NASA Earthdata: https://www.earthdata.nasa.gov/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- USDA NASS: https://www.nass.usda.gov/
- Radiant MLHub on AWS Open Data: https://registry.opendata.aws/radiant-mlhub/

## Routing Rules

- Put remote-sensing-specific resources in `Remote_Sensing/`.
- Put general crop disease and biomass resources in the parent directory.
