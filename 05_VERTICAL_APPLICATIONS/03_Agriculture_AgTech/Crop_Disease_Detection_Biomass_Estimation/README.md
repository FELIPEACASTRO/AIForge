# Crop Disease Detection and Biomass Estimation

This directory covers combined models for crop disease detection and biomass estimation.

## Scope

- Multi-task learning, field imagery, remote sensing, disease severity, vegetation structure, biomass targets, and agronomic decision support.
- Track task balance, labels, sensor, crop, season, metric, and downstream action.

## Reference Links

- PlantVillage dataset: https://github.com/spMohanty/PlantVillage-Dataset
- NASA Earthdata: https://www.earthdata.nasa.gov/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- Papers with Code agriculture tasks: https://paperswithcode.com/area/computer-vision/agriculture

## Routing Rules

- Put disease-only work in plant disease detection.
- Put biomass-only work in biomass estimation folders.
