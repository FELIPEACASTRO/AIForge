# AgTech Multi-Task Transfer Learning

This directory covers multi-task transfer learning for agriculture AI.

## Scope

- Shared representations across crop health, disease, yield, biomass, phenology, weed detection, and remote-sensing tasks.
- Track source tasks, target tasks, shared layers, task weights, datasets, and transfer gains.

## Reference Links

- PyTorch transfer learning tutorial: https://docs.pytorch.org/tutorials/beginner/transfer_learning_tutorial.html
- Hugging Face timm: https://huggingface.co/docs/timm/index
- NASA Earthdata: https://www.earthdata.nasa.gov/
- Radiant MLHub on AWS Open Data: https://registry.opendata.aws/radiant-mlhub/

## Routing Rules

- Put fine-tuning strategies in sibling `Fine_Tuning_Strategies/`.
- Put task-specific outputs in crop subdirectories.
