# Transfer Learning In Agriculture

This directory covers transfer learning for agriculture and AgTech, especially when labeled field data is limited, geography shifts, sensors change, or crop conditions differ.

## Content Map

| Area | Scope |
|---|---|
| Remote sensing transfer | Region, season, crop, sensor, and resolution transfer for satellite and drone imagery. |
| Plant disease transfer | Few-shot disease recognition, cross-crop transfer, domain adaptation, and lab-to-field shift. |
| Biomass and yield transfer | Transferring representations across sites, years, measurement protocols, and crop types. |
| Edge deployment | Distillation and adaptation from large models to field/mobile models. |

## Evaluation Notes

Use out-of-region, out-of-season, out-of-sensor, and out-of-crop tests whenever possible. Record target domain, source domain, labels, and adaptation method.
