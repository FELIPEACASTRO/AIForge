# Launch Documentation

This directory stores launch, public-presence, community, and discoverability assets for AIForge.

## Scope

- GitHub launch checklists, public profile copy, community templates, metadata, wiki/Wikidata notes, and announcement material.
- Keep launch assets factual: cite repository metrics, branch status, source coverage, and validated claims.
- Avoid claiming global leadership without the benchmark file and supporting evidence.
