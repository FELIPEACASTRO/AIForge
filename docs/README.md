# AIForge Documentation

This directory stores repository-level documentation that is not tied to a single AI topic. Use it for navigation, launch notes, audit logs, enrichment plans, publishing metadata, and reproducibility records.

## What Belongs Here

| Area | Use |
|---|---|
| Navigation and discovery | Maps that help readers find material across all five AIForge pillars. |
| Repository operations | Launch, publishing, benchmark, citation, and metadata notes. |
| Enrichment audits | Directory coverage checks, missing README inventories, and batch plans for filling content gaps. |
| Site and search support | Search, sitemap, discoverability, and LLM-facing documentation. |

## Routing Rules

- Put technical AI content in the closest topic directory, not here.
- Put cross-repository reports here when they describe structure, coverage, or maintenance.
- Link back to the exact files or directories that were updated so later batches can continue without guessing.
