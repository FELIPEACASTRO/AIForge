# Resource Catalogs

This directory covers curated catalogs, indexes, awesome lists, registries, benchmarks, and source maps for AI, ML, data, models, prompts, and tooling.

## Scope

- Cross-domain source lists, official portals, benchmark registries, model/dataset indexes, and repo-scale discovery files.
- Catalog entries with topic, authority level, license, update cadence, coverage, and routing destination inside AIForge.

## Reference Links

- Hugging Face models: https://huggingface.co/models
- Hugging Face datasets: https://huggingface.co/datasets
- Papers with Code: https://paperswithcode.com/
- Kaggle datasets: https://www.kaggle.com/datasets
- OpenML: https://www.openml.org/
- UCI Machine Learning Repository: https://archive.ics.uci.edu/

## Routing Rules

- Put global country/source coverage in `../Global_AI_Ecosystem/`.
- Put raw dataset families in `../Datasets/`.
- Put production tools in MLOps or data-engineering folders.
