# Gaming and Reinforcement Learning Datasets

This directory covers datasets and environments for games, reinforcement learning, imitation learning, offline RL, game AI, and interactive agents.

## Scope

- Game replays, trajectories, policies, environment logs, human demonstrations, synthetic episodes, and benchmark tasks.
- Metadata for environment version, action space, observation space, reward function, episode termination, seed policy, and evaluation protocol.
