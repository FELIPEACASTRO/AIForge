# Radiomics Feature Datasets

This directory covers datasets and derived feature tables for radiomics research.

## Scope

- Imaging cohorts, segmentation masks, radiomics feature tables, oncology imaging, reproducibility studies, and clinical outcome labels.
- Track modality, segmentation source, extraction settings, feature definitions, scanner/protocol, and outcome target.

## Reference Links

- The Cancer Imaging Archive: https://www.cancerimagingarchive.net/
- PyRadiomics: https://pyradiomics.readthedocs.io/
- NCI Genomic Data Commons: https://gdc.cancer.gov/
- Medical Segmentation Decathlon: http://medicaldecathlon.com/

## Routing Rules

- Put feature extraction methods in data-engineering radiomics features.
- Put oncology applications in healthcare oncology.
