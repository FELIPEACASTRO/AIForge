# Computer Vision Datasets

This directory covers image datasets for classification, detection, segmentation, OCR, recognition, medical imaging, remote sensing, robotics, and multimodal vision-language tasks.

## Dataset Families

| Family | Examples |
|---|---|
| Classification | ImageNet-style labels, fine-grained classes, domain-specific image classification. |
| Detection | Bounding boxes, object localization, instance detection, safety and surveillance labels. |
| Segmentation | Semantic, instance, panoptic, medical, satellite, and industrial segmentation masks. |
| OCR and documents | Text detection, document layout, handwriting, forms, tables, and charts. |
| Vision-language | Image-caption pairs, VQA, visual instruction tuning, and grounding datasets. |

## Required Metadata

Record license, label source, annotation quality, class balance, resolution, split, benchmark metric, and sensitive-content constraints.
