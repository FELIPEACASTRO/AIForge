# Robotics Datasets

This directory covers datasets for robotics, embodied AI, manipulation, navigation, perception, control, simulation, and physical-world agents.

## Scope

- Robot demonstrations, policies, trajectories, sensor logs, teleoperation, simulation rollouts, manipulation tasks, navigation maps, and multimodal embodied datasets.
- Metadata for robot platform, sensors, action space, control frequency, environment, task definition, safety constraints, and license.

## Reference Links

- Open X-Embodiment dataset: https://robotics-transformer-x.github.io/
- RoboNet: https://www.robonet.wiki/
- Hugging Face LeRobot: https://huggingface.co/docs/lerobot/index
- NVIDIA Isaac Sim: https://docs.isaacsim.omniverse.nvidia.com/
- Papers with Code robotics: https://paperswithcode.com/area/robots
- Kaggle datasets: https://www.kaggle.com/datasets

## Routing Rules

- Put model-policy architectures in robotics or reinforcement-learning model folders.
- Put physical AI/cloud simulation infrastructure in MLOps or cloud-platform sections.
- Put video datasets in `../Video_Datasets/` when video is the primary modality.
