# Cancer Genomics and Multi-Omics Datasets

This directory covers cancer-focused genomics, transcriptomics, proteomics, methylation, imaging-linked omics, and clinical outcome datasets.

## What Belongs Here

- dataset catalogs with provenance, owner, license, access method, schema, and update cadence
- task definitions, splits, leakage risks, benchmark metrics, baselines, and known limitations
- preprocessing notes, quality checks, privacy constraints, and reproducible download references

## Source Links

- [NCI Genomic Data Commons](https://portal.gdc.cancer.gov/)
- [cBioPortal](https://www.cbioportal.org/)
- [TCIA](https://www.cancerimagingarchive.net/)
- [PhysioNet](https://physionet.org/)
- [PubMed](https://pubmed.ncbi.nlm.nih.gov/)
