# Government Open Data Portals

This directory covers government open-data portals relevant to AI and ML.

## Scope

- National, regional, city, statistical, scientific, health, climate, transport, economic, and administrative open datasets.
- Track jurisdiction, license, update cadence, API, file format, and responsible agency.

## Reference Links

- Data.gov: https://data.gov/
- European Data Portal: https://data.europa.eu/
- UK Find Open Data: https://www.data.gov.uk/
- World Bank Data: https://data.worldbank.org/

## Routing Rules

- Put national AI-policy sources in the global AI ecosystem directory.
- Put social-science datasets in social-science dataset folders.
