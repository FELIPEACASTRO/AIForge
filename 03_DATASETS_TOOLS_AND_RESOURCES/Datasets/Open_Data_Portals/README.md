# Open Data Portals

This directory covers public data portals that can feed AI/ML research, analytics, feature engineering, benchmarking, and domain-specific applications.

## Portal Families

| Family | Examples |
|---|---|
| Government portals | Data.gov, EU open data, national statistics, municipal open data, public health, transport, weather. |
| Cloud public datasets | AWS Open Data, Google Cloud public datasets, Azure open datasets, BigQuery public data. |
| Research repositories | Zenodo, Figshare, OSF, Dryad, institutional repositories, DOI-backed datasets. |
| Domain portals | Climate, finance, geospatial, biomedical, agriculture, education, and scientific data portals. |

## Enrichment Standard

Record owner, license, API or download method, update cadence, schema metadata, data quality notes, and whether the portal is suitable for training, evaluation, or feature sourcing.
