# Cloud-Hosted Open Data Portals

This directory covers open datasets hosted by cloud providers.

## Scope

- Public buckets, marketplace datasets, open-data registries, research data, satellite data, biomedical data, and ML-ready public corpora.
- Track provider, region, license, access method, update cadence, data format, and cost implications.

## Reference Links

- AWS Open Data Registry: https://registry.opendata.aws/
- Google Cloud public datasets: https://cloud.google.com/public-datasets
- Azure Open Datasets: https://learn.microsoft.com/en-us/azure/open-datasets/
- NASA Earthdata: https://www.earthdata.nasa.gov/

## Routing Rules

- Put government portals in sibling government folder.
- Put domain datasets in domain-specific dataset folders.
