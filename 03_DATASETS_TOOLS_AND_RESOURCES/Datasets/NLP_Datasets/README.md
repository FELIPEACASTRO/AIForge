# NLP Datasets

This directory covers datasets for natural-language processing, LLM training, instruction tuning, retrieval, evaluation, safety, multilinguality, and domain language tasks.

## Scope

- Corpora for classification, NER, QA, summarization, translation, dialogue, reasoning, RAG, ranking, code, safety, and preference learning.
- Dataset cards with license, language, domain, collection method, deduplication, contamination risk, split design, and benchmark status.
