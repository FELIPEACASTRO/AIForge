# NLP Datasets

> Foundational natural language processing datasets and benchmarks for language modeling, text classification, natural language inference, and question answering, used to train and evaluate LLMs and NLP models.

## Contents

| Item | Description |
| --- | --- |
| [BookCorpus](BookCorpus.md) | Large corpus of free unpublished books widely used for pretraining language models such as BERT and GPT. |
| [C4 (Colossal Clean Crawled Corpus)](C4_(Colossal_Clean_Crawled_Corpus).md) | Cleaned web-crawl corpus derived from Common Crawl, used to pretrain T5 and other large language models. |
| [GLUE Benchmark (General Language Understanding Evaluation)](GLUE_Benchmark_(General_Language_Understanding_Evaluation).md) | Multi-task benchmark suite for evaluating general natural language understanding. |
| [MultiNLI (Multi-Genre Natural Language Inference)](MultiNLI_(Multi-Genre_Natural_Language_Inference).md) | Crowd-sourced textual entailment corpus spanning multiple genres of written and spoken English. |
| [SNLI (Stanford Natural Language Inference) Corpus](SNLI_(Stanford_Natural_Language_Inference)_Corpus.md) | 570K human-labeled sentence pairs for natural language inference / entailment. |
| [SQuAD (Stanford Question Answering Dataset)](SQuAD_(Stanford_Question_Answering_Dataset).md) | Reading-comprehension benchmark of questions posed on Wikipedia passages. |
| [SuperGLUE Benchmark](SuperGLUE_Benchmark.md) | Harder successor to GLUE with more challenging language-understanding tasks. |
| [The Pile (EleutherAI)](The_Pile_(EleutherAI).md) | 825 GiB diverse open-source language-modeling dataset of 22 subsets for training LLMs. |
| [WikiText (Language Modeling Dataset)](WikiText_(Language_Modeling_Dataset).md) | Long-context language modeling corpus of verified Wikipedia articles (WikiText-2 / WikiText-103). |
