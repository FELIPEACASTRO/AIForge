# Time-Series Datasets

This directory covers temporal datasets for forecasting, classification, anomaly detection, control, sensor analytics, finance, medicine, energy, and operations.

## What Belongs Here

- dataset catalogs with provenance, owner, license, access method, schema, and update cadence
- task definitions, splits, leakage risks, benchmark metrics, baselines, and known limitations
- preprocessing notes, quality checks, privacy constraints, and reproducible download references

## Source Links

- [Monash forecasting repository](https://forecastingdata.org/)
- [UCR Time Series Archive](https://www.timeseriesclassification.com/)
- [GluonTS](https://ts.gluon.ai/stable/)
- [sktime](https://www.sktime.net/en/stable/)
- [Nixtla](https://nixtlaverse.nixtla.io/)
