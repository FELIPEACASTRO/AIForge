# Vegetation Indices Datasets

This directory covers datasets and data products that provide vegetation indices or enable vegetation-index computation.

## Scope

- NDVI, EVI, SAVI, NDWI, NBR, phenology products, Landsat, Sentinel, MODIS, VIIRS, and harmonized products.
- Record sensor, resolution, cadence, coverage, atmospheric correction, cloud masking, license, and access method.

## Reference Links

- Google Earth Engine NDVI datasets: https://developers.google.com/earth-engine/datasets/tags/ndvi
- NASA HLS vegetation indices: https://www.earthdata.nasa.gov/news/harmonized-landsat-sentinel-2-project-releases-vegetation-indices
- NASA Earthdata: https://www.earthdata.nasa.gov/
- Copernicus Data Space: https://dataspace.copernicus.eu/
