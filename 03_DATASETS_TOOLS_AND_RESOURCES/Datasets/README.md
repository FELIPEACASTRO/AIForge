# Datasets

This directory catalogs datasets for AI and machine learning by modality, domain, benchmark use, and provenance. It should help a reader choose training, evaluation, validation, and analysis data without mixing source quality levels.

## Modality And Domain Map

| Area | Examples |
|---|---|
| Text and web | NLP datasets, web corpora, instruction data, prompt datasets, Common Crawl derivatives, code corpora. |
| Vision and video | Image classification, detection, segmentation, OCR, video action recognition, multimodal instruction data. |
| Audio and speech | ASR, TTS, music, environmental sound, speaker recognition, emotion recognition. |
| Tabular and classical ML | UCI, OpenML, Kaggle, benchmark suites, business datasets, feature-engineering examples. |
| Time series | Forecasting, anomaly detection, energy, finance, weather, sensor, and industrial datasets. |
| Scientific and biomedical | Genomics, medical imaging, radiology, protein, structural biology, pharmacology, clinical QA. |
| Geospatial and climate | Earth observation, remote sensing, vegetation indices, weather, agriculture, and climate datasets. |
| Robotics and RL | Simulators, trajectories, control benchmarks, embodied AI data, game environments. |

## Source Quality Labels

| Label | Meaning |
|---|---|
| Official | Hosted by a government, university, standards body, company, or original project maintainer. |
| Academic | DOI, paper, benchmark, or lab-hosted dataset with citation metadata. |
| Community | Useful but community-maintained; verify license, provenance, and updates. |
| Synthetic | Generated data; record generator, seed process, evaluation use, and contamination risk. |
| Sensitive | Contains health, biometric, demographic, location, financial, or personal data; record governance constraints. |
