# Agriculture Datasets

This directory covers datasets for agriculture, food systems, crop science, livestock, soil, weather, remote sensing, robotics, and precision agriculture.

## Scope

- Crop yield, crop disease, weed detection, plant phenotyping, biomass, soil, irrigation, weather, satellite, drone, and field-robot data.
- RGB, multispectral, hyperspectral, LiDAR, SAR, weather stations, field sensors, tabular farm records, and geospatial products.
- Dataset notes with license, location, crop type, growing season, sensor, spatial/temporal resolution, and label protocol.

## Reference Links

- Radiant MLHub on AWS Open Data: https://registry.opendata.aws/radiant-mlhub/
- USDA National Agricultural Statistics Service: https://www.nass.usda.gov/
- NASA Earthdata: https://www.earthdata.nasa.gov/
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- Digital agriculture datasets collection: https://github.com/ricber/digital-agriculture-datasets
- Kaggle datasets: https://www.kaggle.com/datasets
