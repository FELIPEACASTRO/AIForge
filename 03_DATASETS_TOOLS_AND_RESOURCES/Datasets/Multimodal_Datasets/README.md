# Multimodal Datasets

This directory covers datasets that combine two or more modalities, such as text-image, image-audio, video-text, document-layout, sensor-fusion, embodied trajectories, and vision-language instruction data.

## Dataset Families

| Family | Examples |
|---|---|
| Vision-language | Image captioning, VQA, OCR, chart QA, document understanding, visual instruction tuning. |
| Video-language | Video QA, action narration, temporal grounding, video captioning, and video instruction data. |
| Audio-language | Speech instruction data, audio captioning, sound event reasoning, and spoken dialogue. |
| Sensor fusion | Autonomous driving, robotics, lidar-camera-radar, wearable sensors, and multimodal health. |
| Document multimodal | PDFs, tables, forms, layout, figures, and scanned document datasets. |

## Required Metadata

Record modality pairs, license, language, annotation method, split, size, evaluation task, and whether the dataset may contain personal or copyrighted content.
