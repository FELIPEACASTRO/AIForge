# Vision-Language Datasets

This directory covers datasets that align images or videos with text.

## Scope

- Image-caption pairs, instruction data, region-text grounding, OCR, document images, visual reasoning, and multimodal conversation data.
- Track source, license, filtering, caption quality, language, safety, and evaluation split.

## Reference Links

- COCO Captions: https://cocodataset.org/#captions-2015
- LAION datasets: https://laion.ai/
- Hugging Face datasets: https://huggingface.co/datasets
- Papers with Code multimodal: https://paperswithcode.com/area/multimodal

## Routing Rules

- Put VQA-specific datasets in `../VQA/`.
- Put multimodal models in model directories.
