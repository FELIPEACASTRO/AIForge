# Audio Multimodal Datasets

This directory covers datasets combining audio with text, video, images, metadata, or sensor streams.

## Scope

- Audio-caption, audio-visual, speech-video, music-text, meeting transcripts, sound-event videos, and multimodal retrieval datasets.
- Track audio format, sample rate, language, transcript quality, video alignment, license, and consent.

## Reference Links

- AudioSet: https://research.google.com/audioset/
- AVA dataset: https://research.google.com/ava/
- Hugging Face audio datasets: https://huggingface.co/datasets?modality=modality:audio
- Papers with Code audio: https://paperswithcode.com/area/audio

## Routing Rules

- Put audio-only datasets in audio dataset folders.
- Put video-audio datasets in audiovisual video dataset folders.
