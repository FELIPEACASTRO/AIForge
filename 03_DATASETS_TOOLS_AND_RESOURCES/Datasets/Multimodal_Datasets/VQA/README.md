# Visual Question Answering Datasets

This directory covers datasets for VQA and visual reasoning.

## Scope

- Image-question-answer, video-question-answer, chart QA, document QA, driving VQA, medical VQA, and multimodal reasoning.
- Track modality, source images/videos, question generation, answer type, license, benchmark split, and evaluation metric.

## Reference Links

- VQA dataset: https://visualqa.org/
- GQA: https://cs.stanford.edu/people/dorarad/gqa/
- DocVQA: https://www.docvqa.org/
- Papers with Code VQA: https://paperswithcode.com/task/visual-question-answering

## Routing Rules

- Put VLM model notes in multimodal model folders.
- Put autonomous-driving VQA in autonomous-driving datasets.
