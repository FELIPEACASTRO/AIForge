# Autonomous Driving Multimodal Datasets

This directory covers multimodal datasets for autonomous driving and driver-assistance AI.

## Scope

- Camera, LiDAR, radar, GPS/IMU, maps, trajectories, driving VQA, scene graphs, weather, and rare-event perception.
- Track sensors, calibration, geography, conditions, annotation tasks, license, and safety-critical slices.

## Reference Links

- nuScenes: https://www.nuscenes.org/
- Waymo Open Dataset: https://waymo.com/open/
- Argoverse: https://www.argoverse.org/
- KITTI: https://www.cvlibs.net/datasets/kitti/

## Routing Rules

- Put VQA driving datasets in VQA if question-answering is primary.
- Put robotics datasets in robotics dataset folders.
