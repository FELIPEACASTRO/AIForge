# Famous Benchmarks

> Landmark computer-vision benchmark datasets that defined modern deep learning, used for training, evaluating, and comparing image classification, object detection, and segmentation models.

## Contents

| Item | Description |
| --- | --- |
| [COCO](COCO.md) | Common Objects in Context: large-scale object detection, segmentation, and captioning dataset with 330K images and 80 categories. |
| [ImageNet](ImageNet.md) | Massive labeled image database organized by the WordNet hierarchy, the benchmark behind the deep learning revolution (ILSVRC). |
| [MNIST](MNIST.md) | Classic handwritten-digit dataset of 70K grayscale images, the canonical "hello world" benchmark for image classification. |
