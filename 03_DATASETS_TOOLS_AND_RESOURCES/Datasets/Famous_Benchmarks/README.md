# Famous Benchmarks

This directory covers widely used AI/ML benchmarks and benchmark families across language, vision, audio, video, robotics, agents, coding, retrieval, safety, and production ML.

## Scope

- Benchmark descriptions, task definitions, datasets, metrics, leaderboards, limitations, contamination risks, and reproducibility notes.
- Keep benchmark claims dated and source-backed; many leaderboards move quickly.

## Reference Links

- Papers with Code: https://paperswithcode.com/
- MLCommons benchmarks: https://mlcommons.org/benchmarks/
- Stanford HELM: https://crfm.stanford.edu/helm/
- SWE-bench: https://www.swebench.com/
- EleutherAI lm-evaluation-harness: https://github.com/EleutherAI/lm-evaluation-harness
- Hugging Face Open LLM Leaderboard: https://huggingface.co/spaces/open-llm-leaderboard/open_llm_leaderboard

## Routing Rules

- Put dataset-specific benchmark material in the relevant dataset modality folder.
- Put model evaluation methodology in `../../../01_AI_FUNDAMENTALS_AND_THEORY/Model_Evaluation/`.
- Put production eval harnesses in MLOps.
