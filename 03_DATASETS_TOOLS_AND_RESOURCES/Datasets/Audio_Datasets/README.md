# Audio Datasets

This directory covers datasets for speech, music, sound events, bioacoustics, environmental audio, audio captioning, speaker recognition, and multimodal audio-language tasks.

## Scope

- ASR, TTS, speaker diarization, keyword spotting, music information retrieval, sound classification, audio generation, and audio-language alignment.
- Metadata for sampling rate, channel count, language, license, speaker consent, recording environment, annotation method, and task splits.

## Reference Links

- Hugging Face audio datasets: https://huggingface.co/datasets?modality=modality:audio
- Mozilla Common Voice: https://commonvoice.mozilla.org/en/datasets
- LibriSpeech: https://www.openslr.org/12/
- AudioSet: https://research.google.com/audioset/
- ESC-50: https://github.com/karolpiczak/ESC-50
- Papers with Code audio tasks: https://paperswithcode.com/area/audio
