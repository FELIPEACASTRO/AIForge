# Cross-Domain Finance Datasets

This directory covers finance datasets that combine market, macroeconomic, alternative, text, graph, geospatial, payments, or business data.

## What Belongs Here

- dataset catalogs with provenance, owner, license, access method, schema, and update cadence
- task definitions, splits, leakage risks, benchmark metrics, baselines, and known limitations
- preprocessing notes, quality checks, privacy constraints, and reproducible download references

## Source Links

- [SEC EDGAR](https://www.sec.gov/search-filings)
- [FRED](https://fred.stlouisfed.org/)
- [World Bank Data](https://data.worldbank.org/)
- [FINRA data](https://www.finra.org/finra-data)
- [Kaggle finance datasets](https://www.kaggle.com/datasets?search=finance)
