# Finance Datasets

This directory covers datasets for banking, fraud, credit risk, financial markets, economics, insurance, payments, crypto, and regulatory analytics.

## Dataset Families

| Family | Examples |
|---|---|
| Market data | Prices, trades, order books, options, futures, FX, crypto, fundamentals, and macro series. |
| Banking and credit | Credit scoring, loan performance, KYC, AML, payments, fraud, and customer behavior. |
| Text and filings | Earnings calls, SEC filings, news, analyst reports, social sentiment, and central-bank text. |
| Risk and insurance | Claims, underwriting, loss events, exposure data, cyber risk, and actuarial datasets. |

## Governance Notes

Record license, survivorship bias, look-ahead bias, timestamp quality, data vendor restrictions, privacy, and whether the dataset can be used for live trading or only research.
