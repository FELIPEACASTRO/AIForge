# Financial Statement Data Sets

## Description
The Financial Statement Data Sets provide numerical information extracted from the financial statements of corporate reports filed with the U.S. Securities and Exchange Commission (SEC). The data is extracted from corporate financial reports filed with the Commission using the eXtensible Business Reporting Language (XBRL). The information is presented unchanged from the "as-filed" financial reports submitted by each registrant. The data is presented in a flattened format to help users analyze and compare corporate disclosure information over time and across registrants.

## Statistics
The data is available from January 2009 through June 2025 and is updated quarterly. The size of the quarterly ZIP files varies, with the most recent files (2023-2025) ranging from approximately 75 MB to 121 MB.

## Features
Numerical financial statement data; XBRL format; quarterly updates; includes the company's Standard Industrial Classification (SIC).

## Use Cases
Financial statement analysis, comparison of company performance, academic research in finance and accounting, development of financial forecasting models, and analysis of market trends.

## Integration
The data sets can be downloaded as quarterly ZIP files directly from the SEC website. Each ZIP file contains the data in a text format that can be imported into databases or analysis tools. The SEC also provides PDF documentation with details on scope, organization, file formats, and table definitions. There are also open-source libraries (for example, in Python) available to simplify the process of working with this data.
