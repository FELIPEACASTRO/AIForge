# Quandl Financial Data (Nasdaq Data Link)

## Description
Quandl, now part of Nasdaq Data Link, is a cloud-based financial intelligence platform that provides access to a vast and diverse collection of financial, economic, and alternative data. The unified platform allows institutional investors, fintechs, and individual users to discover, integrate, and analyze high-quality data to optimize investment and research strategies. Although the name Quandl is still used to refer to its collection of historical data, the infrastructure and access are provided by Nasdaq Data Link.

## Statistics
The Nasdaq Data Link platform (formerly Quandl) indexes millions of data time series from more than 400 sources. The catalog includes more than 250 premium datasets and a vast collection of free and open data. The WIKI Prices dataset, one of the most well-known, contains end-of-day data for 3,000 U.S. companies, but was discontinued in 2018. The platform is used by more than 800,000 global users.

## Features
Access to high-quality financial, economic, and alternative data; time series data and tables (Datatables); RESTful API and Streaming API for real-time data; rigorous data curation (fewer than 1 in 100 datasets analyzed become products); native integration with Python, R, Excel, and SQL; historical and real-time data.

## Use Cases
Market modeling and forecasting; backtesting of quantitative trading strategies; academic research in finance and economics; risk and compliance analysis; development of Machine Learning models for sentiment analysis and stock price forecasting.

## Integration
Access is done through the Nasdaq Data Link API, which requires an API key (free for open data). Nasdaq Data Link offers SDKs and libraries for Python, R, Excel, and SQL, facilitating integration. Data access is done through API calls for time series or tables (Datatables).
