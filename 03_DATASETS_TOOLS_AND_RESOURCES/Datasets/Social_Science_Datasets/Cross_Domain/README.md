# Cross-Domain Social Science Datasets

This directory covers social-science datasets that combine multiple domains.

## Scope

- Economics, education, labor, health, governance, demographics, environment, mobility, public finance, and institutional indicators.
- Track unit of observation, geography, time range, license, sampling design, and bias.

## Reference Links

- World Bank Data: https://data.worldbank.org/
- OECD Data Explorer: https://data-explorer.oecd.org/
- Harvard Dataverse: https://dataverse.harvard.edu/
- ICPSR: https://www.icpsr.umich.edu/sites/icpsr/find-data

## Routing Rules

- Put AI-governance-specific datasets in `../AI_Governance/`.
- Put climate/geospatial social indicators in climate/geospatial datasets.
