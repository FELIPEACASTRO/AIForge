# Video Datasets

This directory covers datasets for action recognition, temporal localization, video question answering, video generation, surveillance, sports analytics, robotics, and multimodal video-language learning.

## Dataset Families

| Family | Examples |
|---|---|
| Action recognition | Kinetics, UCF101, HMDB51, Something-Something, AVA-style activity labels. |
| Video-language | Captioning, video QA, temporal grounding, retrieval, and instruction-following video datasets. |
| Generative video | Text-to-video, image-to-video, editing, style transfer, and controllable generation datasets. |
| Domain video | Medical, sports, traffic, industrial inspection, robotics, agriculture, and security video. |

## Evaluation Notes

Record clip length, frame rate, resolution, label granularity, temporal annotations, splits, licensing, privacy risk, and whether frames are extracted or video-native.
