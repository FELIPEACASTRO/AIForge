# Audiovisual Datasets

This directory covers datasets that combine video with audio, speech, music, sound events, captions, transcripts, or multimodal labels.

## Scope

- Video-audio understanding, audiovisual speech recognition, sound source localization, video captioning, action recognition, event detection, and multimodal retrieval.
- Dataset cards should record license, consent, source platform, clip duration, frame rate, audio format, labels, captions, transcripts, and safety concerns.

## Reference Links

- AudioSet: https://research.google.com/audioset/
- AVA dataset: https://research.google.com/ava/
- VGGSound: https://www.robots.ox.ac.uk/~vgg/data/vggsound/
- Hugging Face video datasets: https://huggingface.co/datasets?modality=modality:video
- Papers with Code video tasks: https://paperswithcode.com/area/computer-vision/video

## Routing Rules

- Put pure video datasets in the parent video dataset directory.
- Put audio-only datasets in `../../Audio_Datasets/`.
- Put video-generation model notes in `../../../../02_LLM_AND_AI_MODELS/Video_Models/`.
