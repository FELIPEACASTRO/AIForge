# Action Recognition Datasets

This directory covers video datasets for action recognition and temporal understanding.

## Scope

- Human actions, activities, sports, surveillance, robotics, driving, multimodal video, and temporal event recognition.
- Track clip source, duration, frame rate, labels, license, train/test split, and evaluation metric.

## Reference Links

- Kinetics dataset benchmark: https://paperswithcode.com/dataset/kinetics-400
- UCF101 benchmark: https://paperswithcode.com/dataset/ucf101
- HMDB51: https://serre-lab.clps.brown.edu/resource/hmdb-a-large-human-motion-database/
- Papers with Code action recognition: https://paperswithcode.com/task/action-recognition-in-videos

## Routing Rules

- Put audiovisual datasets in audiovisual folders.
- Put video model catalogs in video model directories.
