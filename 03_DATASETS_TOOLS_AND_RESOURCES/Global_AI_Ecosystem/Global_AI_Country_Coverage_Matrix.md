# Global AI Country Coverage Matrix

This matrix defines the country universe for AIForge's global AI expansion. It uses 195 independent countries from the DataHub country-codes dataset, which is based on ISO 3166-1 and UN/M49 style fields. Territories and dependent areas can be added in a later appendix, but this file is the "all countries" baseline for national AI coverage.

Source universe: [datasets/country-codes](https://github.com/datasets/country-codes/blob/master/data/country-codes.csv), filtered where `is_independent = Yes`.

## Status Legend

| Status | Meaning |
|---|---|
| S0 seeded | Country is present in the all-country matrix. |
| S1 global-index covered | Country appears in one or more credible global AI indexes. |
| S2 official national source | Official national AI strategy, AI regulator, ministry, or government AI source added. |
| S3 ecosystem sources | Local academic, industry, dataset, startup, compute, safety, and community sources added. |
| S4 verified current | Links and dates checked; at least two independent source families reconciled. |
