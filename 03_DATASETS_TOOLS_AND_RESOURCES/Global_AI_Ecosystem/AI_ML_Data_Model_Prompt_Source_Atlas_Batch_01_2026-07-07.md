## Deferred Or Lower-Trust Source Types

| Source type | Handling rule |
|---|---|
| Public prompt hubs | Useful for inspiration, but mark as community/user-generated unless the prompt is official or reviewed. |
| Leaked system prompts | Do not treat as authoritative reusable prompts; record only as security/research artifacts when legal and ethical context is clear. |
| Marketing model lists | Use only as pointers; verify against official model docs, API docs, model cards, or release notes. |
| Leaderboard snapshots | Capture access date, benchmark version, dataset split, metric, and whether the leaderboard is maintained. |
| Web-scale data corpora | Record license, removal policy, provenance, sensitive-data risk, and copyright concerns before recommending for training. |
