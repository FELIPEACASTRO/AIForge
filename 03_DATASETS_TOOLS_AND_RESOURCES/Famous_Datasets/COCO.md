### Overview

COCO (Common Objects in Context) is a large-scale object detection, segmentation, and captioning dataset. It contains 330K images (>200K labeled), 1.5 million object instances, 80 object categories, and 5 captions per image.

**Source:** https://cocodataset.org/

### Key Features

- **330K images**
- **1.5 million object instances**
- **80 object categories**
- **5 captions per image**
