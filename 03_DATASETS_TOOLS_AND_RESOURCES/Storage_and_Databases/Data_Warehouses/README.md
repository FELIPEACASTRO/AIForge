# Data Warehouses

This directory covers analytical data warehouses used for AI and ML workloads.

## Scope

- Structured analytics, feature source tables, BI, cohort construction, offline training data, semantic layers, and governance.
- Track schema, partitioning, lineage, refresh, access control, and downstream feature use.

## Reference Links

- BigQuery documentation: https://cloud.google.com/bigquery/docs
- Snowflake documentation: https://docs.snowflake.com/
- Amazon Redshift: https://docs.aws.amazon.com/redshift/
- Databricks SQL: https://docs.databricks.com/en/sql/

## Routing Rules

- Put raw/unstructured storage in data lakes.
- Put online features in feature stores.
