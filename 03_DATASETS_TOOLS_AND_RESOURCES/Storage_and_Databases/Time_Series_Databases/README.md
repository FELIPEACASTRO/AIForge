# Time-Series Databases

This directory covers databases optimized for time-series AI data.

## Scope

- Metrics, logs, sensor data, finance series, IoT, monitoring signals, model telemetry, and event streams.
- Track timestamp semantics, retention, downsampling, tags, query latency, and aggregation windows.

## Reference Links

- InfluxDB documentation: https://docs.influxdata.com/
- Timescale documentation: https://docs.timescale.com/
- Prometheus documentation: https://prometheus.io/docs/
- VictoriaMetrics documentation: https://docs.victoriametrics.com/

## Routing Rules

- Put model observability metrics in MLOps observability.
- Put forecasting datasets in time-series dataset/project folders.
