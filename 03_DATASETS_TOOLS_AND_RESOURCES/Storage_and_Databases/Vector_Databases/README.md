# Vector Databases

This directory covers vector stores and retrieval infrastructure for embeddings, semantic search, RAG, recommendation, multimodal retrieval, agent memory, and hybrid search.

## Content Map

| Area | Scope |
|---|---|
| Vector indexes | HNSW, IVF, PQ, diskANN-style indexes, exact search, approximate search, and recall/latency trade-offs. |
| Products and systems | Pinecone, Weaviate, Milvus, Qdrant, pgvector, Elasticsearch/OpenSearch vector search, LanceDB, Chroma, FAISS. |
| RAG infrastructure | Chunking, metadata filters, hybrid retrieval, reranking, multi-vector retrieval, and freshness. |
| Operations | Index build, updates, deletes, backups, tenancy, cost, observability, and evaluation. |

## Evaluation Notes

Track recall, latency, throughput, filter performance, update cost, memory/disk footprint, and retrieval quality on real application queries.
