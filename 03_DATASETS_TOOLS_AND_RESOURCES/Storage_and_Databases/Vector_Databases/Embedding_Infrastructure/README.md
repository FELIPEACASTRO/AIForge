# Embedding Infrastructure

> Vector_Databases covers managed/server DBs (Pinecone/Weaviate/Milvus/Qdrant), but the ANN search libraries and embedding-generation infra that power RAG are absent: FAISS, hnswlib, ScaNN, the sentence-transformers / FlagEmbedding / Infinity embedding-serving stack. FAISS appears only as an incidental mention in CV/dataset docs. This is the engine layer of every RAG and semantic-search system and deserves a dedicated tools subsection.
