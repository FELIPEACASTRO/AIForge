# General Databases

This directory covers general-purpose databases used in AI applications for metadata, operational state, features, user data, application memory, and analytics support.

## Content Map

| Subdirectory | Scope |
|---|---|
| `Relational_Databases/` | PostgreSQL, MySQL, SQL Server, transactional data, metadata, and analytical SQL patterns. |
| `NoSQL_Databases/` | Key-value, document, wide-column, graph-adjacent, and operational stores. |

## AI-Specific Notes

Record workload type, latency, consistency, schema evolution, governance, backup, integration with vector search, and suitability for online inference.
