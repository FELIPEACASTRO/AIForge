# Graph Databases

This directory covers graph databases used in AI, knowledge graphs, fraud detection, recommendations, and relationship analytics.

## Scope

- Property graphs, RDF graphs, graph queries, graph embeddings, graph retrieval, lineage, and graph-backed AI applications.
- Track graph model, schema, query language, update cadence, scale, and integration with ML.
