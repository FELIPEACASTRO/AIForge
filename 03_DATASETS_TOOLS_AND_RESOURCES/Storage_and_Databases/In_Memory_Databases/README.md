# In-Memory Databases

This directory covers in-memory data systems used by AI applications.

## Scope

- Caches, session memory, online features, low-latency lookups, queues, counters, and ephemeral state for agents and model serving.
- Track latency, TTL, eviction, persistence, replication, and data sensitivity.

## Reference Links

- Redis documentation: https://redis.io/docs/latest/
- KeyDB documentation: https://docs.keydb.dev/
- Dragonfly documentation: https://www.dragonflydb.io/docs
- Memcached documentation: https://docs.memcached.org/

## Routing Rules

- Put generic key-value stores in key-value store directories.
- Put vector search in vector databases.
