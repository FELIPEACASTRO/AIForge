# Document Databases

This directory covers document databases used in AI applications.

## Scope

- JSON/document storage, metadata stores, conversation state, retrieval metadata, application records, and semi-structured AI data.
- Track schema, index strategy, consistency, access control, retention, and query patterns.

## Reference Links

- MongoDB documentation: https://www.mongodb.com/docs/
- Couchbase documentation: https://docs.couchbase.com/
- Apache CouchDB: https://docs.couchdb.org/
- Azure Cosmos DB: https://learn.microsoft.com/en-us/azure/cosmos-db/

## Routing Rules

- Put vector retrieval in vector database folders.
- Put key-value caching in key-value store directories.
