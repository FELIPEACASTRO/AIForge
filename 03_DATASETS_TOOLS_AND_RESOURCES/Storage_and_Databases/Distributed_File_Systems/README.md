# Distributed File Systems

This directory covers distributed file systems for large-scale AI and data workloads.

## Scope

- Shared datasets, model checkpoints, training data, logs, high-throughput reads, and cluster storage.
- Track filesystem, mount model, throughput, consistency, replication, security, and failure recovery.

## Reference Links

- HDFS: https://hadoop.apache.org/docs/stable/hadoop-project-dist/hadoop-hdfs/HdfsDesign.html
- CephFS: https://docs.ceph.com/en/latest/cephfs/
- Lustre: https://www.lustre.org/
- Azure Managed Lustre: https://learn.microsoft.com/en-us/azure/azure-managed-lustre/

## Routing Rules

- Put object storage in storage infrastructure/object folders.
- Put data lake architecture in data lakes.
