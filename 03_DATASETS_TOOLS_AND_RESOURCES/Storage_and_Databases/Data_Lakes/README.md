# Data Lakes

This directory covers data lakes and lakehouse patterns for AI and ML.

## Scope

- Raw, curated, and feature-ready data zones; open table formats; object storage; metadata catalogs; governance; and reproducible training datasets.
- Track storage path, table format, schema evolution, partitioning, access control, and lifecycle.

## Reference Links

- Delta Lake: https://docs.delta.io/
- Apache Iceberg: https://iceberg.apache.org/docs/latest/
- Apache Hudi: https://hudi.apache.org/docs/overview/
- Databricks data lakes overview: https://www.databricks.com/discover/data-lakes

## Routing Rules

- Put data warehouses in data warehouse folders.
- Put object storage in storage infrastructure.
