# Storage Infrastructure

This directory covers storage infrastructure used by AI and data systems.

## Scope

- Object storage, block/file storage, tiering, encryption, lifecycle policies, metadata catalogs, access control, and cost management.
- Track storage class, durability, region, versioning, retention, lineage, security, and disaster recovery.

## Reference Links

- Amazon S3: https://docs.aws.amazon.com/s3/
- Google Cloud Storage: https://cloud.google.com/storage/docs
- Azure Blob Storage: https://learn.microsoft.com/en-us/azure/storage/blobs/
- MinIO: https://min.io/docs/minio/linux/index.html
