# Data Streaming Platforms

This directory covers streaming data platforms used in AI/ML pipelines.

## Scope

- Event streams, CDC, logs, online features, inference events, telemetry, and real-time training/evaluation data.
- Track topic/schema, event time, watermark, retention, replay, exactly-once assumptions, and consumer ownership.

## Reference Links

- Apache Kafka: https://kafka.apache.org/documentation/
- Apache Flink: https://nightlies.apache.org/flink/flink-docs-stable/
- Apache Beam: https://beam.apache.org/documentation/
- Redpanda: https://docs.redpanda.com/

## Routing Rules

- Put batch pipelines in data-pipeline directories.
- Put online features in feature-store directories.
