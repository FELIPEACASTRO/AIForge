# Data Streaming

This directory covers streaming data systems for AI and analytics.

## Scope

- Event streams, logs, sensor data, clickstreams, online features, inference events, CDC, and streaming transformations.
- Track event schema, event time, ordering, replay, retention, latency, and consumer ownership.

## Reference Links

- Apache Kafka: https://kafka.apache.org/documentation/
- Apache Flink: https://nightlies.apache.org/flink/flink-docs-stable/
- Apache Beam: https://beam.apache.org/documentation/
- Redpanda: https://docs.redpanda.com/

## Routing Rules

- Put platform-specific notes in `platform/`.
- Put batch pipelines in data-pipeline directories.
