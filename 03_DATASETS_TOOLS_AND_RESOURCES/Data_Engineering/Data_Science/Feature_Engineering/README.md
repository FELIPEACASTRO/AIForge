# Data Science Feature Engineering

This directory covers feature-engineering resources aimed at exploratory and applied data-science workflows.

## Scope

- Tabular transformations, leakage checks, encoding, scaling, aggregations, temporal windows, and baseline-ready feature sets.
- Track target definition, split strategy, transformation scope, missingness, and validation impact.

## Reference Links

- scikit-learn preprocessing: https://scikit-learn.org/stable/modules/preprocessing.html
- scikit-learn feature selection: https://scikit-learn.org/stable/modules/feature_selection.html
- pandas documentation: https://pandas.pydata.org/docs/

## Routing Rules

- Put production feature stores in feature-store directories.
- Put theory in the AI fundamentals feature-engineering directory.
