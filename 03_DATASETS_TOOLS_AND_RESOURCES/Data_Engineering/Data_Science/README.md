# Data Science

This directory covers data-science resources inside the data-engineering pillar.

## Scope

- Exploratory analysis, feature engineering, statistical checks, notebooks, public datasets, and analysis-ready datasets.
- Track dataset source, target, assumptions, transformations, metrics, and reproducibility.

- Jupyter documentation: https://docs.jupyter.org/
