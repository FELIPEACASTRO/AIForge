# Public Datasets for Analysis

This directory covers public datasets used for analysis examples and reproducible data-science workflows.

## Scope

- Open datasets for EDA, visualization, data-quality checks, benchmark analysis, and tutorial notebooks.
- Track license, source, version, update date, schema, target variables, and known bias.

## Reference Links

- Data.gov: https://data.gov/
- Kaggle datasets: https://www.kaggle.com/datasets
- OpenML: https://www.openml.org/
- UCI Machine Learning Repository: https://archive.ics.uci.edu/

## Routing Rules

- Put full dataset catalogs in the datasets pillar.
- Put analysis methods in the parent data-analysis directory.
