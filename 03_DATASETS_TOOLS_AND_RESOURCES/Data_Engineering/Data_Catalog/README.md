# Data Catalog

This directory covers catalogs that make AI/ML datasets discoverable and governable.

## Scope

- Metadata, ownership, schema, lineage, quality signals, business glossary, access requests, and dataset search.
- Track catalog tool, metadata model, ingestion source, data owner, and freshness.

## Reference Links

- DataHub: https://datahubproject.io/docs/
- OpenMetadata: https://docs.open-metadata.org/
- Amundsen GitHub: https://github.com/amundsen-io/amundsen
- CKAN: https://docs.ckan.org/

## Routing Rules

- Put data quality rules in data-quality directories.
- Put open data portals in dataset portal folders.
