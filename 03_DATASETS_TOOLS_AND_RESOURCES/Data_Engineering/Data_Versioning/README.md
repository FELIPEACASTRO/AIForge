# Data Versioning

This directory covers versioning for datasets, features, labels, and AI artifacts.

## Scope

- Dataset snapshots, table versions, object versioning, label versions, data diffing, lineage, and reproducible rebuilds.
- Track version ID, source, schema, checksum, split, license, and downstream model compatibility.

## Reference Links

- DVC: https://dvc.org/doc
- lakeFS: https://docs.lakefs.io/
- Delta Lake: https://docs.delta.io/
- Apache Iceberg: https://iceberg.apache.org/docs/latest/

## Routing Rules

- Put storage architecture in data lakes or storage infrastructure.
- Put feature lifecycle in feature-store directories.
