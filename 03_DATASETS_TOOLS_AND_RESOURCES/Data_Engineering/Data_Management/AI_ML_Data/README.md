# AI/ML Data Management

This directory covers management of datasets used in AI and ML systems.

## Scope

- Data ownership, catalogs, lineage, versioning, quality, contracts, governance, privacy, retention, and access control.
- Track dataset owner, schema, license, sensitivity, lineage, version, and usage constraints.

## Reference Links

- OpenMetadata: https://docs.open-metadata.org/
- DataHub: https://datahubproject.io/docs/
- Great Expectations: https://docs.greatexpectations.io/
- DVC: https://dvc.org/doc

## Routing Rules

- Put storage engines in storage/database directories.
- Put data quality in data-quality directories.
