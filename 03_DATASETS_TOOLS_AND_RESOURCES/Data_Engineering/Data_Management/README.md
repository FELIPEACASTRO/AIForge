# Data Management

This directory covers governance and lifecycle management for AI/ML data.

## Scope

- Ownership, lineage, catalogs, contracts, access, privacy, quality, retention, licensing, versioning, and auditability.
- Track data owner, schema, sensitivity, license, lineage, retention, and downstream model usage.

## Reference Links

- DataHub: https://datahubproject.io/docs/
- OpenMetadata: https://docs.open-metadata.org/
- DVC: https://dvc.org/doc
- Great Expectations: https://docs.greatexpectations.io/

## Routing Rules

- Put AI/ML-specific data governance in `AI_ML_Data/`.
- Put storage engines in storage/database directories.
