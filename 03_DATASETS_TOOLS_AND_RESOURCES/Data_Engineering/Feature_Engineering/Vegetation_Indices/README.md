# Vegetation Indices Features

This directory covers feature engineering with vegetation indices for remote sensing and agriculture AI.

## Scope

- NDVI, EVI, SAVI, NDWI, NBR, red-edge indices, phenology curves, temporal aggregates, and texture features.
- Track sensor, bands, atmospheric correction, cloud masking, temporal window, spatial resolution, and target leakage risk.

## Reference Links

- Google Earth Engine NDVI datasets: https://developers.google.com/earth-engine/datasets/tags/ndvi
- NASA HLS vegetation indices: https://www.earthdata.nasa.gov/news/harmonized-landsat-sentinel-2-project-releases-vegetation-indices
- Google Earth Engine datasets: https://developers.google.com/earth-engine/datasets
- USDA NASS: https://www.nass.usda.gov/
