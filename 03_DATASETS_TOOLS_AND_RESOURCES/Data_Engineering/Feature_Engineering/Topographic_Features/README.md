# Topographic Features

This directory covers terrain-derived features for geospatial, climate, agriculture, and environmental AI.

## Scope

- Elevation, slope, aspect, curvature, terrain ruggedness, flow accumulation, hillshade, watershed features, and spatial context.
- Track DEM source, resolution, projection, processing method, and leakage risks.

## Reference Links

- NASA SRTM: https://www.earthdata.nasa.gov/sensors/srtm
- USGS 3DEP: https://www.usgs.gov/3d-elevation-program
- Google Earth Engine terrain algorithms: https://developers.google.com/earth-engine/apidocs/ee-algorithms-terrain
- Copernicus DEM on AWS Open Data: https://registry.opendata.aws/copernicus-dem/

## Routing Rules

- Put climate/geospatial datasets in dataset folders.
- Put agriculture applications in AgTech folders.
