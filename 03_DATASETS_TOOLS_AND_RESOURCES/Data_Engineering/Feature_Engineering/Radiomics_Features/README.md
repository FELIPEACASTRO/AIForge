# Radiomics Features

This directory covers engineered medical-imaging features used in radiomics workflows.

## Scope

- Shape, first-order, texture, wavelet, filter-derived, lesion-level, organ-level, and longitudinal imaging features.
- Preprocessing, segmentation dependency, binning, resampling, harmonization, scanner effects, reproducibility, and feature stability.
