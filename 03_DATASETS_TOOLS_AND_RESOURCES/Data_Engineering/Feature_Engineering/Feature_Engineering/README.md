# Feature Engineering Resources

This directory stores general feature-engineering resources inside the data-engineering tree.

## Scope

- Feature definitions, reusable transformations, leakage-safe preprocessing, feature documentation, and validation examples.
- Track source columns, target, transformation, training/serving parity, and evaluation impact.

## Reference Links

- scikit-learn preprocessing: https://scikit-learn.org/stable/modules/preprocessing.html
- scikit-learn pipelines: https://scikit-learn.org/stable/modules/compose.html
- pandas documentation: https://pandas.pydata.org/docs/

## Routing Rules

- Put theoretical material in the AI fundamentals feature-engineering directory.
- Put production feature serving in feature-store directories.
