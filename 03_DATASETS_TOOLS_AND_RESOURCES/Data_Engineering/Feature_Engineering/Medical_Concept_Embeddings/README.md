# Medical Concept Embeddings

This directory covers vector representations of medical concepts, codes, notes, ontologies, and biomedical entities.

## Scope

- SNOMED, ICD, LOINC, RxNorm, UMLS, PubMed entities, EHR concepts, clinical notes, and patient-level embeddings.
- Track vocabulary, corpus, model, embedding dimension, update date, license, and evaluation task.

## Reference Links

- UMLS: https://www.nlm.nih.gov/research/umls/
- SNOMED CT: https://www.snomed.org/
- LOINC: https://loinc.org/
- RxNorm: https://www.nlm.nih.gov/research/umls/rxnorm/
- PubMed: https://pubmed.ncbi.nlm.nih.gov/

## Routing Rules

- Put clinical NLP applications in healthcare clinical NLP.
- Put vector storage in vector database folders.
