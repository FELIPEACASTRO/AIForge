# Histology Features

This directory covers engineered features for histology and pathology AI.

## Scope

- Tissue morphology, nuclei features, texture, stain normalization, tile embeddings, spatial features, and weakly supervised slide-level signals.
- Track stain, scanner, magnification, tile size, annotation level, aggregation method, and validation cohort.

## Reference Links

- The Cancer Imaging Archive: https://www.cancerimagingarchive.net/
- OpenSlide: https://openslide.org/
- HistomicsTK: https://digitalslidearchive.github.io/HistomicsTK/
- MONAI documentation: https://monai-dev.readthedocs.io/

## Routing Rules

- Put oncology applications in healthcare oncology.
- Put general radiomics in radiomics feature directories.
