# Feature Engineering

This directory covers transforming raw data into useful machine-learning signals across tabular, text, image, geospatial, biomedical, financial, and time-series domains.

## Content Map

| Subdirectory | Scope |
|---|---|
| `Feature_Engineering/` | General feature construction, transformations, interactions, and domain-specific techniques. |
| `Vegetation_Indices/` | NDVI-style spectral indices, remote-sensing features, texture features, and agriculture signals. |
| `Topographic_Features/` | Elevation, slope, aspect, terrain derivatives, and geospatial covariates. |
| `Radiomics_Features/` | Medical imaging features, PyRadiomics, lesion features, and clinical imaging descriptors. |
| `Medical_Concept_Embeddings/` | Biomedical ontologies, clinical concept embeddings, and medical text representations. |
| `Phenotypic_Features_Extraction/` | Phenotyping pipelines, biomedical signals, and patient-level derived features. |
| `Features_Histologicas/` | Pathology and histology feature extraction. |

## Enrichment Standard

Feature pages should include raw input type, transformation, leakage risk, scaling/normalization, model fit, and validation method.
