# Phenotypic Feature Extraction

This directory covers extraction of phenotypic features for biomedical, clinical, agricultural, and biological AI.

## Scope

- Clinical phenotypes, plant phenotypes, imaging phenotypes, EHR-derived cohorts, temporal features, and ontology-backed features.
- Track phenotype definition, data source, extraction logic, ontology, validation, and bias.

## Reference Links

- Human Phenotype Ontology: https://hpo.jax.org/
- OHDSI phenotyping: https://ohdsi.github.io/TheBookOfOhdsi/Cohorts.html
- PlantCV: https://plantcv.readthedocs.io/
- PhysioNet: https://physionet.org/

## Routing Rules

- Put clinical NLP extraction in healthcare clinical NLP.
- Put plant disease phenotyping in agriculture folders.
