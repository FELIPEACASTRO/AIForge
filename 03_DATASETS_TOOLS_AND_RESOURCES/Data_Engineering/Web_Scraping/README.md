# Web Scraping for AI Data

This directory covers web scraping and extraction workflows for AI datasets.

## Scope

- Crawling, scraping, parsing, extraction, robots.txt, rate limits, deduplication, provenance, and dataset cleaning.
- Track source URL, license/terms, crawl date, parser, fields, consent/privacy risk, and refresh cadence.

## Reference Links

- Scrapy documentation: https://docs.scrapy.org/
- Beautiful Soup documentation: https://www.crummy.com/software/BeautifulSoup/bs4/doc/
- Playwright documentation: https://playwright.dev/docs/intro
- Common Crawl: https://commoncrawl.org/

## Routing Rules

- Put resulting datasets in the dataset pillar.
- Put API-based ingestion in API integration or data source directories.
