# Data Preprocessing Augmentation

This directory covers augmentation as part of preprocessing pipelines.

## Scope

- Image, text, audio, tabular, time-series, geospatial, graph, and multimodal augmentation before model training.
- Track invariance assumption, label preservation, split safety, parameter ranges, and validation impact.

## Reference Links

- Albumentations: https://albumentations.ai/docs/
- TorchVision transforms: https://pytorch.org/vision/stable/transforms.html
- TensorFlow image augmentation: https://www.tensorflow.org/tutorials/images/data_augmentation

## Routing Rules

- Put domain-specific augmentation in the relevant modality folder.
- Put training strategy notes in AI fundamentals.
