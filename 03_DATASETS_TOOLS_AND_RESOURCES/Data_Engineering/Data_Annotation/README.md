# Data Annotation

This directory covers annotation processes and tooling for AI datasets.

## Scope

- Image, video, text, audio, 3D, tabular, geospatial, biomedical, and multimodal annotation.
- Track annotation tool, instructions, label schema, reviewer process, agreement, and edge cases.

## Reference Links

- CVAT: https://docs.cvat.ai/
- Label Studio: https://labelstud.io/guide/
- doccano: https://doccano.github.io/doccano/
- FiftyOne: https://docs.voxel51.com/

## Routing Rules

- Put label QA and storage in data labeling.
- Put annotation datasets in the relevant dataset modality folder.
