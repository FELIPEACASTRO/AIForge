# Data Labeling

This directory covers labeling workflows for AI datasets.

## Scope

- Annotation guidelines, label taxonomies, labeling tools, consensus, QA, active learning, weak supervision, and label versioning.
- Track task, label schema, annotator qualifications, agreement, QA process, and license/consent.

## Reference Links

- Label Studio: https://labelstud.io/guide/
- CVAT: https://docs.cvat.ai/
- Snorkel: https://snorkel.ai/
- Prodigy: https://prodi.gy/docs

## Routing Rules

- Put annotation tooling in data annotation.
- Put label storage/versioning in data versioning or management.
