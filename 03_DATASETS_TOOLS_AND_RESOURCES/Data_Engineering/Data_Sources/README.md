# Data Sources

This directory covers source systems used to collect AI/ML data.

## Scope

- APIs, databases, files, object stores, event streams, crawlers, sensors, open-data portals, warehouses, and third-party feeds.
- Track owner, access method, schema, license, sensitivity, cadence, reliability, and downstream usage.

## Reference Links

- OpenAPI Specification: https://spec.openapis.org/oas/latest.html
- Common Crawl: https://commoncrawl.org/
- Data.gov: https://data.gov/
- Kaggle datasets: https://www.kaggle.com/datasets

## Routing Rules

- Put ingestion pipelines in data pipelines or ETL folders.
- Put source catalogs in resource catalog folders.
