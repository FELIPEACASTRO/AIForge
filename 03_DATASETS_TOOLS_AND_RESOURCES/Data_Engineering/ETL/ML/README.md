# ETL for Machine Learning

This directory covers ETL patterns for machine-learning and AI datasets.

## Scope

- Extract, transform, load, validation, partitioning, backfills, data contracts, lineage, and reproducible dataset builds.
- Track source, schema, owner, cadence, transformation logic, validation, and downstream models.

## Reference Links

- dbt documentation: https://docs.getdbt.com/docs/introduction
- Apache Airflow: https://airflow.apache.org/docs/
- Dagster: https://docs.dagster.io/
- Great Expectations: https://docs.greatexpectations.io/

## Routing Rules

- Put orchestration in data pipelines or workflow orchestration.
- Put analysis notebooks in data-analysis directories.
