# Feature Stores

This directory covers feature-store resources from a data-engineering point of view.

## Scope

- Offline/online stores, materialization, point-in-time joins, feature definitions, validation, freshness, and reuse.
- Track entity keys, timestamps, transformation logic, backfill, online serving, owner, and deprecation policy.
