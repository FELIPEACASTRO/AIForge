## Why this section exists

A "world's-largest ML index" must point not only to models and datasets, but to the **research infrastructure** that produces them. arXiv is the hub, but reproducibility (Papers with Code), open metadata graphs (OpenAlex), and modern reading tools (alphaXiv, HF Papers) are now core to a working ML research loop.
