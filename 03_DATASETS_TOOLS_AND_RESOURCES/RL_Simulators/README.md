# Reinforcement Learning Simulators

This directory covers simulation environments, benchmark suites, control tasks, robotics worlds, game environments, and embodied-agent platforms.

## What Belongs Here

- dataset catalogs with provenance, owner, license, access method, schema, and update cadence
- task definitions, splits, leakage risks, benchmark metrics, baselines, and known limitations
- preprocessing notes, quality checks, privacy constraints, and reproducible download references

## Evidence To Track

- Source URL, publication or update date, license, owner, and access conditions.
